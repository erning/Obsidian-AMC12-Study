# AMC12 2013-B Q10

![[assets/problems/2013-B/Q10.png]]

## Problem

Alex has $75$ red tokens and $75$ blue tokens. At one booth, Alex gives two red tokens and receives a silver token and a blue token. At another booth, Alex gives three blue tokens and receives a silver token and a red token. Alex continues exchanging until no more exchanges are possible. How many silver tokens does Alex have at the end?

## Answer

**E.** $103$

## Solution

Let $m$ be the number of red-booth exchanges and $n$ the number of blue-booth exchanges. The final token counts are
$$R=75-2m+n,qquad B=75-3n+m,$$
and the silver count is $m+n$.

At the end, neither exchange is possible, so $R\le1$ and $B\le2$. Equivalently,
$$2m-n\ge74,qquad3n-m\ge73.$$
The smallest feasible pair is $(m,n)=(59,44)$, for which both inequalities are tight. Thus the number of silver tokens is
$$m+n=59+44=103.$$

## Knowledge points

- Invariants and state variables
- Systems of linear inequalities
- Integer optimization

## Key idea

Track the two exchangeable token types and minimize the number of exchanges needed to block both booths.

## Common mistakes

- Counting silver tokens as $m-n$
- Reversing which token each booth consumes
