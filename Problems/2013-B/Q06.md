# AMC12 2013-B Q06

![[assets/problems/2013-B/Q06.png]]

## Problem

Real numbers $x$ and $y$ satisfy
$$x^2+y^2=10x-6y-34.$$
What is $x+y$?

## Answer

**B.** $2$

## Solution

Move all terms to the left and complete the square:
$$
x^2-10x+25+y^2+6y+9=0,
$$
so
$$
(x-5)^2+(y+3)^2=0.
$$
Both squares must be zero. Hence $x=5$, $y=-3$, and
$$x+y=2.$$

## Knowledge points

- Completing the square
- Nonnegative sums

## Key idea

A sum of two real squares equals zero only when both squares vanish.

## Common mistakes

- Solving the equation as if it described a nontrivial circle
- Completing $y^2-6y$ with the wrong sign
