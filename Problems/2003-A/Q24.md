# AMC12 2003-A Q24

![[assets/problems/2003-A/Q24.png]]

## Problem

If $a \ge b \gt 1,$ what is the largest possible value of $\log_a(a/b) + \log_b(b/a)?$

(A) $-2$
(B) $0$
(C) $2$
(D) $3$
(E) $4$

## Answer

**B.** $0$

## Solution

Expand:
$$
 \begin{aligned} &\log_a\dfrac ab+\log_b\dfrac ba \\ &\quad {}=(1-\log_a b)+(1-\log_b a) \\ &\quad {}=2-\left(\log_a b+\log_b a\right). \end{aligned}
$$

Let $c=\log_a b\gt0.$ Since $c+\dfrac1c\ge2$ by AM-GM, the expression is at most $0.$

Equality holds when $c=1,$ that is, when $a=b,$ so the largest value is $0.$

Thus, the correct answer is **B**.

## Key idea

- Simplify to $2-\log_a b-\log_b a$
- With $c=\log_a b\gt0,$ use $c+\dfrac1c\ge2$

- Logarithm
- Am-gm inequality

## Common mistakes

- Match each algebraic or geometric quantity to the exact wording of the problem.
- Check units, endpoint conditions, and whether the requested value is a count, length, area, or probability.
