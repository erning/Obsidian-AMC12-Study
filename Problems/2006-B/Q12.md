# AMC12 2006-B Q12

![[assets/problems/2006-B/Q12.png]]

## Problem

The parabola $y = ax^2 + bx + c$ has vertex $(p, p)$ and $y$-intercept $(0, -p),$ where $p \neq 0.$ What is $b?$

(A) $-p$
(B) $0$
(C) $2$
(D) $4$
(E) $p$

## Answer

**D.** $4$

## Solution

The vertex form is $y = a(x - p)^2 + p.$

At $x = 0,$ $y = ap^2 + p = -p,$ so $ap^2 = -2p$ and $a = -\dfrac{2}{p}.$

Expanding, $y = a x^2 - 2ap\, x + ap^2 + p,$ so $b = -2ap = -2\left(-\dfrac{2}{p}\right)p = 4.$

Thus, the correct answer is **D**.

## Knowledge points

- Parabola
- Quadratic

## Key idea

Use the vertex/roots or the quadratic equation to describe the parabola.
