# AMC12 2004-B Q17

![[assets/problems/2004-B/Q17.png]]

## Problem

For some real numbers $a$ and $b,$ the equation
$$
8x^3 + 4ax^2 + 2bx + a = 0
$$
 has three distinct positive roots. If the sum of the base-$2$ logarithms of the roots is $5,$ what is the value of $a?$

(A) $-256$
(B) $-64$
(C) $-8$
(D) $64$
(E) $256$

## Answer

**A.** $-256$

## Solution

The sum of the base-$2$ logarithms is $\log_2(r_1 r_2 r_3) = 5,$ so $r_1 r_2 r_3 = 2^5 = 32.$ By Vieta's formulas on $8x^3 + 4ax^2 + 2bx + a,$ the product of the roots is $-\dfrac{a}{8}.$ Thus $-\dfrac{a}{8} = 32,$ giving $a = -256.$

Thus, the correct answer is **A**.

## Knowledge points

- Vieta's formulas
- Logarithm

## Key idea

Use Vieta's formulas instead of solving the polynomial directly.
