# AMC12 2004-B Q16

![[assets/problems/2004-B/Q16.png]]

## Problem

A function $f$ is defined by $f(z) = i\overline{z},$ where $i = \sqrt{-1}$ and $\overline{z}$ is the complex conjugate of $z.$ How many values of $z$ satisfy both $|z| = 5$ and $f(z) = z?$

(A) $0$
(B) $1$
(C) $2$
(D) $4$
(E) $8$

## Answer

**C.** $2$

## Solution

Writing $z = x + iy,$ we get $f(z) = i(x - iy) = y + ix.$ Setting $f(z) = z$ gives $y = x,$ which is a line through the origin. The condition $|z| = 5$ is a circle, and a line through the center meets the circle in $2$ points.

Thus, the correct answer is **C**.
