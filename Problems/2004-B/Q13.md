# AMC12 2004-B Q13

## Problem

If $f(x) = ax + b$ and $f^{-1}(x) = bx + a$ with $a$ and $b$ real, what is the value of $a + b?$

(A) $-2$
(B) $-1$
(C) $0$
(D) $1$
(E) $2$

## Answer

**A.** $-2$

## Solution

Since $f(f^{-1}(x)) = x,$ we have $a(bx + a) + b = x.$ Matching terms gives $ab = 1$ and $a^2 + b = 0.$ Then $b = 1/a$ and $a^2 + 1/a = 0,$ so $a^3 = -1,$ giving $a = -1$ and $b = -1.$ Thus $a + b = -2.$

Thus, the correct answer is **A**.
