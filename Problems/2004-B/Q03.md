# AMC12 2004-B Q03

![[assets/problems/2004-B/Q03.png]]

## Problem

If $x$ and $y$ are positive integers for which $2^x 3^y = 1296,$ what is the value of $x + y?$

(A) $8$
(B) $9$
(C) $10$
(D) $11$
(E) $12$

## Answer

**A.** $8$

## Solution

Factoring, $1296 = 6^4 = 2^4 \cdot 3^4.$ Matching exponents gives $x = 4$ and $y = 4,$ so $x + y = 8.$

Thus, the correct answer is **A**.

## Knowledge points

- Prime factorization
- Exponent

## Key idea

Factor the number and use its prime exponents.
