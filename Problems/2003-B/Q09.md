# AMC12 2003-B Q09

![[assets/problems/2003-B/Q09.png]]

## Problem

Let $f$ be a linear function for which $f(6) - f(2) = 12.$ What is $f(12) - f(2)?$

(A) $12$
(B) $18$
(C) $24$
(D) $30$
(E) $36$

## Answer

**D.** $30$

## Solution

The slope of $f$ is
$$
 \frac{f(6) - f(2)}{6 - 2} = \frac{12}{4} = 3.
$$

Therefore
$$
 f(12) - f(2) = 3(12 - 2) = 30.
$$

Thus, the correct answer is **D**.

## Key idea

- A linear function has a constant rate of change (slope)
- The slope is $\dfrac{f(6) - f(2)}{6 - 2},$ and the answer is that slope times $12 - 2$

- Slope
- Linear equation

## Common mistakes

- Check units, endpoint conditions, and whether the requested quantity is a count, length, area, or probability.
