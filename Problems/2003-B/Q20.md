# AMC12 2003-B Q20

![[assets/problems/2003-B/Q20.png]]

## Problem

Part of the graph of $f(x) = ax^3 + bx^2 + cx + d$ is shown. What is $b?$

(A) $-4$
(B) $-2$
(C) $0$
(D) $2$
(E) $4$

## Answer

**B.** $-2$

## Solution

The graph passes through $(-1, 0),$ $(1, 0),$ and $(0, 2).$ So $f(0) = d = 2.$

Adding
$$
 \begin{aligned} &f(1) + f(-1) \\ &= (a + b + c + d) \\ &\quad {}+ (-a + b - c + d) \\ &= 2b + 2d = 0, \end{aligned}
$$
 so $b = -d = -2.$

Thus, the correct answer is **B**.

## Key idea

- Read off $f(-1) = 0,$ $f(1) = 0,$ and $f(0) = 2$ from the graph
- Adding $f(1) + f(-1)$ eliminates $a$ and $c,$ leaving $2b + 2d = 0$
