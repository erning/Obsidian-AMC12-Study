# AMC12 2005-B Q02

![[assets/problems/2005-B/Q02.png]]

## Problem

A positive number $x$ has the property that $x\%$ of $x$ is $4.$ What is $x?$

(A) $2$
(B) $4$
(C) $10$
(D) $20$
(E) $40$

## Answer

**D.** $20$

## Solution

The statement translates to
$$
 \dfrac{x}{100}\cdot x = 4,
$$
 so $x^2 = 400.$

Since $x$ is positive, $x = 20.$

Thus, the correct answer is **D**.

## Knowledge points

- Percentage
- Quadratic

## Key idea

Represent the percentage change using the same original quantity.

## Common mistakes

- Do not use a formula before identifying exactly which quantities the statement fixes.
- Keep track of units, signs, and endpoint conditions throughout the calculation.
