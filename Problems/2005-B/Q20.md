# AMC12 2005-B Q20

![[assets/problems/2005-B/Q20.png]]

## Problem

Let $a, b, c, d, e, f, g$ and $h$ be distinct elements in the set
$$
 \{-7, -5, -3, -2, 2, 4, 6, 13\}.
$$
 What is the minimum possible value of
$$
 \begin{aligned} &(a + b + c + d)^2 \\ &\quad {}+ (e + f + g + h)^2? \end{aligned}
$$

(A) $30$
(B) $32$
(C) $34$
(D) $40$
(E) $50$

## Answer

**C.** $34$

## Solution

The elements sum to $8.$ If $a + b + c + d = x,$ then $e + f + g + h = 8 - x,$ so
$$
 x^2 + (8 - x)^2 = 2(x - 4)^2 + 32.
$$

This is minimized when $x = 4,$ giving $32.$ But $13$ must lie in one group, and no three of the remaining elements add with $13$ to make $4$ (that would need three of them summing to $-9,$ which is impossible here). So $x = 4$ is unattainable and $(x - 4)^2 \ge 1.$

The minimum is $2(1) + 32 = 34,$ achieved for instance by $\{-7, -5, 2, 13\}$ (sum $3$) and $\{-3, -2, 4, 6\}$ (sum $5$).

Thus, the correct answer is **C**.
