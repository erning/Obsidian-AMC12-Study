# AMC12 2015-B Q03

![[assets/problems/2015-B/Q03.png]]

## Problem

Isaac has written down one integer two times and another integer three times. The sum of the five numbers is $100$, and one of the numbers is $28$. What is the other number?

## Answer

**A.** $8$

## Solution

Let $x$ be the number written twice and $y$ the number written three times. Then

$$2x+3y=100.$$

If $y=28$, then

$$2x=100-3(28)=16,$$

so $x=8$. If $x=28$, then

$$3y=100-2(28)=44,$$

which is impossible because $44$ is not divisible by $3$. Therefore the other number is $8$.

## Knowledge points

- Linear Diophantine equations
- Divisibility

## Key idea

Test whether the known value occupies the two-copy or three-copy position.

## Common mistakes

- Assuming $28$ is automatically the number written three times
- Dividing $44$ by $3$ and accepting a noninteger answer
