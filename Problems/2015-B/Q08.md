# AMC12 2015-B Q08

![[assets/problems/2015-B/Q08.png]]

## Problem

What is the value of

$$\left(625^{\log_5 2015}\right)^{1/4}?$$

## Answer

**D.** $2015$

## Solution

Since $625=5^4$,

$$\left(625^{\log_5 2015}\right)^{1/4}
=\left((5^4)^{\log_5 2015}\right)^{1/4}
=\left(5^{4\log_5 2015}\right)^{1/4}.$$

Taking the fourth root cancels the exponent $4$:

$$5^{\log_5 2015}=2015.$$

## Knowledge points

- Logarithm identities
- Exponent laws

## Key idea

Rewrite $625$ as $5^4$, then use $5^{\log_5 x}=x$.

## Common mistakes

- Reading the fourth-root exponent as applying only to $625$
- Replacing $\log_5 2015$ with $\log_{2015}5$
