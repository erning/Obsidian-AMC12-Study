# AMC12 2015-A Q12

![[assets/problems/2015-A/Q12.png]]

## Problem

The parabolas $y=ax^2-2$ and $y=4-bx^2$ meet the axes in four points forming a kite of area $12$. Find $a+b$.

## Answer

**B.** $1.5$

## Solution

The vertical diagonal has length $6$. If the horizontal intercepts are $\pm t$, the other diagonal has length $2t$. Thus
$$\frac12(6)(2t)=12,$$
so $t=2$. Then $4a=2$ and $4b=4$, giving $a=1/2,b=1$, so $a+b=1.5$.

## Knowledge points

- Parabola intercepts
- Kite area

## Key idea

Use the two coordinate-axis diagonals of the kite.

## Common mistakes

- Using the distance between the parabolas instead of intercepts
- Forgetting the factor $1/2$
