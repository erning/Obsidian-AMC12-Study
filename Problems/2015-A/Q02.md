# AMC12 2015-A Q02

![[assets/problems/2015-A/Q02.png]]

## Problem

Two sides of a triangle are $20$ and $15$. Which listed number cannot be its perimeter?

## Answer

**E.** $72$

## Solution

If the third side is $s$, the Triangle Inequality gives
$$5<s<35.$$
The perimeter is $35+s$, so it lies strictly between $40$ and $70$. Only $72$ is impossible.

## Knowledge points

- Triangle Inequality
- Bounds

## Key idea

Bound the unknown side before considering the perimeter.

## Common mistakes

- Allowing equality in the Triangle Inequality
- Checking only the lower bound
