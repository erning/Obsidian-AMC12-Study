# AMC12 2015-A Q11

![[assets/problems/2015-A/Q11.png]]

## Problem

A radius-$2$ circle and radius-$3$ circle have all common tangent lines drawn. How many different values of the number of tangents are possible?

## Answer

**D.** $5$

## Solution

Depending on position, the circles can be nested (0 tangents), internally tangent (1), intersecting (2), externally tangent (3), or disjoint (4). Thus $k$ can be $0,1,2,3,4$, giving $5$ values.

## Knowledge points

- Common tangents to circles
- Geometric cases

## Key idea

Classify circle positions by center distance.

## Common mistakes

- Counting only external tangents
- Assuming the circles must intersect
