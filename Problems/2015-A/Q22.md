# AMC12 2015-A Q22

![[assets/problems/2015-A/Q22.png]]

## Problem

Let $S(n)$ count A/B sequences of length $n$ with no run longer than $3$. Find $S(2015)\bmod12$.

## Answer

**D.** $8$

## Solution

A valid sequence ends with a run of length $1,2,$ or $3$, so
$$S(n)=S(n-1)+S(n-2)+S(n-3).$$
Computing the recurrence modulo $3$ and $4$ gives $S(2015)\equiv2\pmod3$ and $S(2015)\equiv0\pmod4$. The number modulo $12$ is therefore $8$.

## Knowledge points

- Recurrence relations
- Modular periodicity

## Key idea

Compute the recurrence in the two coprime moduli $3$ and $4$.

## Common mistakes

- Using Fibonacci initial values
- Applying the recurrence only over the integers
