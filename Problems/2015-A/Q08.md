# AMC12 2015-A Q08

![[assets/problems/2015-A/Q08.png]]

## Problem

A rectangle has length-to-width ratio $4:3$ and diagonal $d$. Find $k$ if its area is $kd^2$.

## Answer

**C.** $\frac{12}{25}$

## Solution

Let the sides be $4t$ and $3t$. The diagonal is $5t=d$, so
$$[R]=12t^2=12\left(\frac d5\right)^2=\frac{12}{25}d^2.$$

## Knowledge points

- Pythagorean triples
- Area scaling

## Key idea

Use the $3$-$4$-$5$ ratio.

## Common mistakes

- Using $4d$ and $3d$ as side lengths
- Confusing area ratio with side ratio
