# AMC12 2015-A Q14

![[assets/problems/2015-A/Q14.png]]

## Problem

Find $a$ if $\frac1{\log_2a}+\frac1{\log_3a}+\frac1{\log_4a}=1$.

## Answer

**D.** $24$

## Solution

By change of base,
$$\frac1{\log_ba}=\log_ab.$$
The equation becomes
$$\log_a2+\log_a3+\log_a4=\log_a24=1,$$
so $a=24$.

## Knowledge points

- Change of base
- Logarithm identities

## Key idea

Convert reciprocals of logarithms into logarithms with the base $a$.

## Common mistakes

- Treating $1/\log_ba$ as $\log_b(1/a)$
- Combining logs with different bases
