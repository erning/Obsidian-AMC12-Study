# AMC12 2015-A Q13

![[assets/problems/2015-A/Q13.png]]

## Problem

A 12-team round-robin awards 2 points for a win and 1 for a draw. Which listed statement about scores is not always true?

## Answer

**E.** The highest score need not be at least $12$.

## Solution

There are $\binom{12}{2}=66$ games, and each contributes $2$ total points, so the total score is $132$. If every game is a draw, every team scores $11$, so the highest score need not be at least $12$. The other statements follow from parity, total score, and the fact that two teams cannot both score $0$.

## Knowledge points

- Tournament score invariants
- Parity

## Key idea

Find a valid extreme example to disprove the universal statement.

## Common mistakes

- Assuming every tournament has a winner
- Confusing total score with maximum score
