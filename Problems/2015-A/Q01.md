# AMC12 2015-A Q01

![[assets/problems/2015-A/Q01.png]]

## Problem

Evaluate $(2^0-1+5^2+0)^{-1}\times5$.

## Answer

**C.** $\frac15$

## Solution

Inside the parentheses,
$$2^0-1+5^2+0=1-1+25=25.$$
Therefore
$$25^{-1}\times5=\frac5{25}=\frac15.$$

## Knowledge points

- Exponents
- Negative powers

## Key idea

Simplify the base expression before applying the exponent $-1$.

## Common mistakes

- Interpreting $2^0$ as $0$
- Forgetting that $x^{-1}=1/x$
