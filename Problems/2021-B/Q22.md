# AMC12 2021-B Q22

![[assets/problems/2021-B/Q22.png]]

## Problem

Arjun and Beth play a game in which they take turns removing one brick or two adjacent bricks from one wall among several walls of bricks. The player who removes the last brick wins. For which starting configuration is there a strategy that guarantees a win for Beth?

The choices are $(6,1,1)$, $(6,2,1)$, $(6,2,2)$, $(6,3,1)$, and $(6,3,2)$.

## Answer

**B.** $(6,2,1)$

## Solution

For a wall of length $n$, let $g(n)$ be its Grundy number. Directly from the possible moves,

$$g(1)=1,\quad g(2)=2,\quad g(3)=3,\quad g(4)=1,\quad g(5)=4,\quad g(6)=3.$$

For several independent walls, the position is losing for the player to move exactly when the xor of the Grundy numbers is $0$. The configuration $(6,2,1)$ has

$$g(6)\oplus g(2)\oplus g(1)=3\oplus2\oplus1=0.$$

Thus it is a winning starting position for Beth, who moves second.

## Knowledge points

- Sprague-Grundy theory
- Nim-sum / bitwise xor

## Key idea

Compute the Grundy value for each wall length and use xor to identify the second-player position.

## Common mistakes

- Using ordinary addition instead of xor.
- Confusing a position with xor $0$ as winning for the first player.
