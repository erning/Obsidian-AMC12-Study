# AMC12 2021-B Q01

![[assets/problems/2021-B/Q01.png]]

## Problem

How many integer values of $x$ satisfy $|x|<3\pi$?

## Answer

**D.** $19$

## Solution

Since $3\pi\approx9.42$, the integer condition is

$$-9.42<x<9.42.$$

Therefore $x$ can be any integer from $-9$ through $9$, inclusive. The count is

$$9-(-9)+1=19.$$

## Knowledge points

- Absolute value inequalities
- Counting consecutive integers

## Key idea

Estimate the irrational endpoint only enough to identify the nearest integers.

## Common mistakes

- Counting $-9$ through $9$ as $18$ values instead of $19$.
- Including $10$, even though $10>3\pi$.
