# AMC12 2021-B Q25

![[assets/problems/2021-B/Q25.png]]

## Problem

Let $S$ be the set of lattice points in the coordinate plane whose coordinates are integers between $1$ and $30$, inclusive. Exactly $300$ points in $S$ lie on or below a line with equation $y=mx$. The possible values of $m$ lie in an interval of length $\frac ab$, where $a$ and $b$ are relatively prime positive integers. What is $a+b$?

## Answer

**E.** $85$

## Solution

For a fixed integer $x$, the number of allowed $y$-values on or below the line is

$$\min(30,\lfloor mx\rfloor).$$

When the count is $300$, the line is high enough that the first $10$ columns are unsaturated while the remaining columns are saturated. The boundary values occur when the line passes through the lattice points determining the last change in this count:

$$\frac23\le m\le\frac{19}{28}.$$

The interval length is

$$\frac{19}{28}-\frac23=\frac{57-56}{84}=\frac1{84}.$$

Hence $a=1$, $b=84$, and $a+b=85$.

## Knowledge points

- Lattice-point counting
- Floor functions
- Boundary analysis

## Key idea

The count changes only when the line crosses a lattice point, so determine the two adjacent critical slopes rather than integrate a continuous area.

## Common mistakes

- Counting points strictly below the line instead of on or below it.
- Replacing the floor function by $mx$ and losing the endpoint behavior.
