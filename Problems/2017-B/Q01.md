# AMC12 2017-B Q01

![[assets/problems/2017-B/Q01.png]]

## Problem

Kymbrea has 30 comic books and adds 2 per month. LaShawn has 10 and adds 6 per month. When will LaShawn have twice as many as Kymbrea?

## Answer

**E.** $25$

## Solution

After $m$ months, the two collections contain $30+2m$ and $10+6m$ books. Set the second equal to twice the first:
$$10+6m=2(30+2m).$$
Thus $2m=50$, so $m=25$.

## Knowledge points

- Linear equations
- Translating a rate problem into algebra

## Key idea

Write both quantities at the same time parameter before comparing them.

## Common mistakes

- Reversing which collection should be doubled
- Using the initial difference instead of the monthly rates
