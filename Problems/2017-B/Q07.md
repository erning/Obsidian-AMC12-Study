# AMC12 2017-B Q07

![[assets/problems/2017-B/Q07.png]]

## Problem

Find the least period of $\cos(\sin x)$.

## Answer

**B.** $\pi$

## Solution

Since $\sin(x+\pi)=-\sin x$ and cosine is even,
$$\cos(\sin(x+\pi))=\cos(-\sin x)=\cos(\sin x).$$
Thus $\pi$ is a period. It is least because the function reaches $1$ exactly when $\sin x=0$, and those points are spaced $\pi$ apart.

## Knowledge points

- Periodic functions
- Even and odd function properties

## Key idea

The outer cosine removes the sign change caused by shifting the inner sine by $\pi$.

## Common mistakes

- Assuming composition preserves the period $2\pi$
- Concluding that $\pi/2$ is a period without testing the maxima
