# AMC12 2017-B Q20

![[assets/problems/2017-B/Q20.png]]

## Problem

Choose $x,y$ independently and uniformly from $(0,1)$. Find the probability that $\lfloor\log_2x\rfloor=\lfloor\log_2y\rfloor$.

## Answer

**D.** $\frac13$

## Solution

For $nge1$,
$$\lfloor\log_2x\rfloor=-n$$
exactly when
$$2^{-n}\le x<2^{-(n-1)}.$$
The interval has length $2^{-n}$. The probability that both variables fall in this same interval is $4^{-n}$. Summing,
$$\sum_{n=1}^{\infty}\frac1{4^n}=\frac{1/4}{1-1/4}=\frac13.$$

## Knowledge points

- Geometric series
- Continuous probability and interval lengths

## Key idea

Partition $(0,1)$ into dyadic intervals on which the floor value is constant.

## Common mistakes

- Using interval lengths $2^{-(n-1)}$
- Summing probabilities for different intervals incorrectly
