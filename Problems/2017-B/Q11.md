# AMC12 2017-B Q11

![[assets/problems/2017-B/Q11.png]]

## Problem

A positive integer is monotonous if its digits are strictly increasing or strictly decreasing, or if it has one digit. Count all monotonous positive integers.

## Answer

**B.** $1524$

## Solution

An increasing number is determined by a nonempty subset of ${1,\ldots,9}$, giving
$$2^9-1=511.$$
A decreasing number is determined by a subset of ${0,\ldots,9}$, except $\varnothingarnothing$ and ${0}$ because of leading zero:
$$2^{10}-2=1022.$$
The nine one-digit numbers were counted twice, so the total is
$$511+1022-9=1524.$$

## Knowledge points

- Counting by subsets
- Inclusion-exclusion

## Key idea

A strictly monotone digit string is uniquely determined by the set of digits it uses.

## Common mistakes

- Allowing a leading zero
- Forgetting to remove the overlap of one-digit numbers
