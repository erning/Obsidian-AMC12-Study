# AMC12 2017-B Q17

![[assets/problems/2017-B/Q17.png]]

## Problem

A biased coin has $P(H)=2/3$ and $P(T)=1/3$. Game A wins if three tosses are all equal. Game B wins if tosses 1 and 2 match and tosses 3 and 4 match. Compare the two winning probabilities.

## Answer

**D.** Game A is more likely by $\frac2{81}$.

## Solution

For Game A,
$$P_A=\left(\frac23\right)^3+\left(\frac13\right)^3=\frac{9}{27}=\frac13.$$
A matching pair has probability
$$\left(\frac23\right)^2+\left(\frac13\right)^2=\frac59.$$
Thus
$$P_B=\left(\frac59\right)^2=\frac{25}{81}.$$
Since $\frac13=\frac{27}{81}$, Game A exceeds Game B by
$$\frac{27-25}{81}=\frac2{81}.$$

## Knowledge points

- Independent events
- Bernoulli probabilities

## Key idea

Compute the probability of one matching pair, then square it for Game B.

## Common mistakes

- Treating the two matching pairs as one three-toss event
- Forgetting the all-tails case in Game A
