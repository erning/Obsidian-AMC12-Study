# AMC12 2017-B Q19

![[assets/problems/2017-B/Q19.png]]

## Problem

Let $N=123456789101112\cdots4344$. Find $N\bmod45$.

## Answer

**C.** $9$

## Solution

The last digit gives $N\equiv4\pmod5$. The digit sum is divisible by $9$, so $N\equiv0\pmod9$.

The number $N-9$ is divisible by $9$ and ends in $5$, so it is also divisible by $5$. Therefore $N-9$ is divisible by $45$, and
$$N\equiv9\pmod{45}.$$

## Knowledge points

- Modular arithmetic
- Divisibility by $5$ and $9$

## Key idea

Find a convenient number congruent to $N$ modulo both $5$ and $9$.

## Common mistakes

- Using only the last digit and concluding the remainder is $4$
- Forgetting that divisibility by $45$ requires both $5$ and $9$
