# AMC12 2017-B Q05

![[assets/problems/2017-B/Q05.png]]

## Problem

For the data set with $Q_1=33$ and $Q_3=43$, how many values are outliers under the $1.5$ IQR rule?

## Answer

**B.** $1$

## Solution

The interquartile range is
$$operatorname{IQR}=43-33=10.$$
The outlier cutoffs are
$$33-1.5(10)=18,qquad43+1.5(10)=58.$$
Only $6$ is below $18$, and no value exceeds $58$. Hence there is one outlier.

## Knowledge points

- Quartiles
- Interquartile range and outliers

## Key idea

Compute the two cutoffs before checking the data values.

## Common mistakes

- Using the median instead of the quartiles
- Including values equal to a cutoff as outliers
