# AMC12 2017-B Q08

![[assets/problems/2017-B/Q08.png]]

## Problem

For a rectangle, the short-to-long side ratio equals the long-side-to-diagonal ratio. Find the square of the short-to-long ratio.

## Answer

**C.** $\frac{\sqrt5-1}{2}$

## Solution

Let the short side be $x$, the long side $y$, and set
$$r=\frac{x^2}{y^2}.$$
The condition gives
$$\frac{x}{y}=\frac{y}{\sqrt{x^2+y^2}}.$$
Squaring and using $r=x^2/y^2$,
$$r=\frac1{r+1},qquad r^2+r-1=0.$$
The positive root is
$$r=\frac{\sqrt5-1}{2}.$$

## Knowledge points

- Pythagorean theorem
- Algebraic equations involving ratios

## Key idea

Square the ratio condition so the diagonal becomes $x^2+y^2$.

## Common mistakes

- Confusing the requested squared ratio with $x/y$
- Kee\ping the negative quadratic root
