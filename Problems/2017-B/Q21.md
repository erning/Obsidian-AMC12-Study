# AMC12 2017-B Q21

![[assets/problems/2017-B/Q21.png]]

## Problem

Seven different integer scores lie between $91$ and $100$, inclusive. The seventh score is $95$, and every running average is an integer. Find the sixth score.

## Answer

**E.** $100$

## Solution

Let the total sum be $S$. Since the final average is an integer,
$$Sin{658,665,672,679}.$$
The first six scores have integer average, so $S-95$ is divisible by $6$. This forces $S=665$.

Thus the first six scores sum to $570$. The first five scores have integer average, so their sum is divisible by $5$. Hence the sixth score is divisible by $5$. It cannot be $95$ because scores are different, so the only possible score is $100$.

## Knowledge points

- Divisibility constraints
- Integer averages

## Key idea

Use the final total and the six-score total to narrow the sum before using distinctness.

## Common mistakes

- Forgetting that all seven scores are distinct
- As\suming the sixth score is determined only from the final average
