# AMC12 2017-B Q03

![[assets/problems/2017-B/Q03.png]]

## Problem

Nonzero real numbers $x,y$ satisfy
$$\frac{3x+y}{x-3y}=-2.$$
Find
$$\frac{x+3y}{3x-y}.$$

## Answer

**D.** $2$

## Solution

Cross-multiply:
$$3x+y=-2(x-3y)=-2x+6y.$$
Hence $5x=5y$, so $x=y$. Substituting,
$$\frac{x+3y}{3x-y}=\frac{4y}{2y}=2.$$

## Knowledge points

- Rational equations
- Homogeneous expressions

## Key idea

The given equation determines the ratio $x:y$, so set $x=y$ after simplifying.

## Common mistakes

- Dividing by a quantity before checking it is nonzero
- Substituting numerical values for $x$ and $y$ unnecessarily
