# AMC12 2017-B Q02

![[assets/problems/2017-B/Q02.png]]

## Problem

Given $0<x<1$, $-1<y<0$, and $1<z<2$, which expression is necessarily positive?

## Answer

**E.** $y+z$

## Solution

Because $y>-1$ and $z>1$,
$$y+z> -1+1=0.$$
The other expressions are not always positive. For example, values close to $x=1$, $y=-1$, and $z=1$ make the first four choices negative.

## Knowledge points

- Inequalities
- Strict bounds

## Key idea

Combine the strongest lower bounds directly.

## Common mistakes

- Treating open inequalities as if equality were allowed
- As\suming a positive square is larger than the magnitude of the negative term
