# AMC12 2017-B Q24

![[assets/problems/2017-B/Q24.png]]

## Problem

A right-angle quadrilateral has $\triangle ABC\sim\triangle BCD$ and an interior point $E$ with $\triangle ABC\sim\triangle CEB$. If $[AED]=17[CEB]$, find $AB/BC$.

## Answer

**D.** $2+\sqrt5$

## Solution

Set $BC=1$ and $AB=r$. Place
$$C=(0,0),\quad B=(0,1),\quad A=(r,1),\quad D=(1/r,0).$$
From $\triangle ABC\sim\triangle CEB$, the coordinates of $E$ satisfy
$$x=\frac{r}{1+r^2},qquad y=\frac{r^2}{1+r^2}.$$
Compute the areas using the determinant formula. The condition
$$[AED]=17[CEB]$$
simplifies to
$$r^4-18r^2+1=0.$$
Thus
$$r^2=9+4\sqrt5=(2+\sqrt5)^2.$$
Since $r>1$,
$$r=2+\sqrt5.$$

## Knowledge points

- Similar right triangles
- Coordinate geometry
- Area by determinants

## Key idea

Normalize $BC=1$, use similarity to locate $E$, and reduce the area condition to one equation in $r$.

## Common mistakes

- Choosing the negative square root
- Using the wrong correspondence in the similarity statements
