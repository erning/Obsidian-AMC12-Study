# AMC12 2025-A Q04

![[assets/problems/2025-A/Q04.png]]

## Problem

Agnes writes the following four statements on a blank piece of paper.

1. At least one of these statements is true.
2. At least two of these statements are true.
3. At least two of these statements are false.
4. At least one of these statements is false.

Each statement is either true or false. How many false statements did Agnes write?

**Choices:** (A) 0 &nbsp;&nbsp; (B) 1 &nbsp;&nbsp; (C) 2 &nbsp;&nbsp; (D) 3 &nbsp;&nbsp; (E) 4

## Answer

- Correct choice: **(B) 1**

## Solution

Let $T$ be the number of true statements. Then the four statements assert, respectively,

$$
T\ge 1,\qquad T\ge 2,\qquad T\le 2,\qquad T\le 3.
$$

The value of $T$ must equal the number of statements that are true.

Try $T=3$. The first, second, and fourth statements are true, while the third statement $T\le2$ is false. This produces exactly 3 true statements, so it is consistent.

No other value works:

- $T=0$ makes the first statement false, contradicting $T=0$.
- $T=1$ makes only the first statement true, but the fourth statement is also true.
- $T=2$ makes the first, third, and fourth statements true, giving 3 true statements.
- $T=4$ makes the third statement false, so all four cannot be true.

Therefore there are $4-3=\boxed{1}$ false statements.

## Knowledge points

- Translating verbal conditions into inequalities
- Self-referential logic
- Consistency checking by casework

## Key idea

Do not decide each statement independently. First let $T$ represent the total number of true statements; each statement can then be written as a condition on $T$.

## Common mistakes

- Treating “at least two are false” as $T\ge2$; it means $4-T\ge2$, or $T\le2$.
- Counting the statements that are true without checking whether the count equals the assumed value of $T$.
- Forgetting that the statements refer to the complete set of four statements.
