# AMC12 2023-A Q15

![[assets/problems/2023-A/Q15.png]]

## Problem

Usain runs across a $100$-meter by $30$-meter rectangular field by zigzagging between the two long sides, as shown. What is the angle shown in the diagram?

## Answer

- Correct choice: **(A) $\arccos\frac56$**

## Solution

Each crossing has width $30$. The zigzag consists of four crossings, so the total path length is
$$
4\cdot\frac{30}{\sin\theta}=\frac{120}{\sin\theta}.
$$
The total horizontal advance is
$$
4\cdot30\cot\theta=120\cot\theta=100.
$$
Thus the ratio of horizontal advance to total path length is
$$
\cos\theta=\frac{100}{120}=\frac56.
$$
Therefore $\theta=\boxed{\arccos\frac56}$.

> Note: the step "$\cos\theta=100/120$" is not justified as written — $120\cot\theta=100$ gives only $\cot\theta=\tfrac56$, which is not $\cos\theta$. Whether this holds depends on how the diagram defines $\theta$; please verify against the figure and the official solution.
