# AMC12 2023-A Q13

![[assets/problems/2023-A/Q13.png]]

## Problem

In a table-tennis tournament every pair of players plays one game. There are twice as many right-handed players as left-handed players. Left-handed players win $40\%$ more games than right-handed players. How many games are played?

## Answer

- Correct choice: **(B) $36$**

## Solution

Let the number of left-handed players be $L$, so there are $2L$ right-handed players and
$$
\binom{3L}{2}
$$
games. If right-handed players win $R$ games, left-handed players win $\frac75R$, so the total is $\frac{12}{5}R$. Thus the total number of games must be divisible by $12$. Testing:
$$
\binom32=3,\qquad \binom62=15,\qquad \binom92=36.
$$
The first possible total is $\boxed{36}$, achievable with $21$ left-handed wins and $15$ right-handed wins.

## Key idea

Express the total games and total wins using the number of left-handed players.
