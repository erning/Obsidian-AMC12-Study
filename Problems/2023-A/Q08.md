# AMC12 2023-A Q08

![[assets/problems/2023-A/Q08.png]]

## Problem

A student’s average after $n$ quizzes is $m$. If the next quiz score is $11$, the average becomes $m+1$. If the next three quiz scores are all $11$, the average becomes $m+2$. What is $m$?

## Answer

- Correct choice: **(D) $7$**

## Solution

The first condition gives
$$
\frac{mn+11}{n+1}=m+1,
$$
so $m+n=10$. The second gives
$$
\frac{mn+33}{n+3}=m+2,
$$
which simplifies to $3m+2n=27$. Substituting $n=10-m$:
$$
3m+2(10-m)=27,
$$
so $\boxed{m=7}$.

## Knowledge points

- Averages
- Linear equations

## Key idea

Write the total points before and after each update.

## Common mistakes

- Using $m+1$ for the three-quiz update.
