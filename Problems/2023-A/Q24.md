# AMC12 2023-A Q24

![[assets/problems/2023-A/Q24.png]]

## Problem

Let $K$ be the number of sequences $A_1,A_2,\ldots,A_n$, where $1\le n\le10$, of subsets of $\{1,2,\ldots,10\}$ satisfying $A_{i-1}\subseteq A_i$ for $2\le i\le n$. What is the units digit of $K$?

## Answer

- Correct choice: **(C) $5$**

## Solution

Fix $n$. Each element has a first appearance in one of the $n$ sets, or never appears. Thus each of the $10$ elements has $n+1$ choices, giving $(n+1)^{10}$ sequences. Therefore
$$
K=\sum_{n=1}^{10}(n+1)^{10}=\sum_{k=2}^{11}k^{10}.
$$
Modulo $10$, the terms have residues
$$
4,9,6,5,6,9,4,1,0,1,
$$
whose sum is $45$. The units digit is $\boxed{5}$.

- Combinatorics

## Key idea

Encode each element by the first set in which it appears.

## Common mistakes

- Counting arbitrary set sequences without enforcing nesting.
