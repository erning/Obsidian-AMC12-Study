# AMC12 2004-A Q16

![[assets/problems/2004-A/Q16.png]]

## Problem

The set of all real numbers $x$ for which
$$
\small \log_{2004}(\log_{2003}(\log_{2002}(\log_{2001} x)))
$$
 is defined is $\{x \mid x \gt c\}.$ What is the value of $c?$

(A) $0$
(B) $2001^{2002}$
(C) $2002^{2003}$
(D) $2003^{2004}$
(E) $2001^{2002^{2003}}$

## Answer

**B.** $2001^{2002}$

## Solution

The expression is defined if and only if $\log_{2003}(\log_{2002}(\log_{2001} x)) \gt 0,$ that is, $\log_{2002}(\log_{2001} x) \gt 1.$

This holds if and only if $\log_{2001} x \gt 2002,$ which is equivalent to $x \gt 2001^{2002}.$

Therefore $c = 2001^{2002}.$

Thus, the correct answer is **B**.

- Inequality

## Key idea

Apply logarithm rules and simplify the condition.
