# AMC12 2004-A Q14

![[assets/problems/2004-A/Q14.png]]

## 题目

三个实数构成首项为 $9$ 的等差数列。若在第二项加上 $2$、在第三项加上 $20$，所得三个数构成等比数列。该等比数列第三项的最小可能值是多少？

(A) $1$
(B) $4$
(C) $36$
(D) $49$
(E) $81$

## 答案

**A.** $1$

## 解答

该等差数列为 $9$、$9 + d$、$9 + 2d$。加数后，等比数列为 $9$、$11 + d$、$29 + 2d$。

等比条件给出 $(11 + d)^2 = 9(29 + 2d)$，化简得 $d^2 + 4d - 140 = 0$，故 $d = 10$ 或 $d = -14$。

对应的第三项 $29 + 2d$ 为 $49$ 和 $1$，故最小可能值为 $1$。

因此，正确答案是 **A**。

## 知识点

- 等差数列
- 等比数列
- 二次方程

## 关键思路

利用公差不变及前 $n$ 项和公式。

## 常见错误

- 在确认题目给定的量之前，不要急于套用公式。
- 计算过程中始终注意单位、符号和端点条件。
