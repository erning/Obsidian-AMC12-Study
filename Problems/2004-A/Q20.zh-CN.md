# AMC12 2004-A Q20

![[assets/problems/2004-A/Q20.png]]

## 题目

独立且随机地从 $0$ 到 $1$ 之间选取数 $a$、$b$，令 $c$ 为它们的和。将 $a$、$b$、$c$ 分别四舍五入到最接近的整数，结果为 $A$、$B$、$C$。$A + B = C$ 的概率是多少？

(A) $\dfrac{1}{4}$
(B) $\dfrac{1}{3}$
(C) $\dfrac{1}{2}$
(D) $\dfrac{2}{3}$
(E) $\dfrac{3}{4}$

## 答案

**E.** $\dfrac{3}{4}$

## 解答

将选择表示为单位正方形中的点 $(a, b)$。$a$ 和 $b$ 小于 $\tfrac12$ 时分别四舍五入为 $0$，否则为 $1$；而 $c = a + b$ 的舍入分界值为 $\tfrac12$ 和 $\tfrac32$。

在下列区域中有 $A + B = C$：若 $a + b \lt \tfrac12$，则 $A = B = C = 0$；若 $a,b$ 中恰有一个不小于 $\tfrac12$ 且 $a + b \lt \tfrac32$，则该变量舍入为 $1 = C$；若 $a + b \ge \tfrac32$，则 $A = B = 1$、$C = 2$。

这些区域由两个面积各为 $\tfrac18$ 的角落三角形和两个中央带状区域组成，总面积为 $\tfrac34$。正方形面积为 $1$，所以概率为 $\tfrac34$。

因此，正确答案是 **E**。

## 关键思路

将随机选择表示为区域，并取面积比。

## 常见错误

- 在确认题目给定的量之前，不要急于套用公式。
- 计算过程中始终注意单位、符号和端点条件。
