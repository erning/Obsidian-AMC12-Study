# AMC12 2011-B Q19

![[assets/problems/2011-B/Q19.png]]

## Problem

A lattice point in an $xy$-coordinate system is any point $(x, y)$ where both $x$ and $y$ are integers. The graph of $y=mx+2$ passes through no lattice point with $0 \lt x \le 100$ for all $m$ such that $\frac{1}{2} \lt m \lt a.$ What is the maximum possible value of $a?$

(A) $\frac{51}{101}$
(B) $\frac{50}{99}$
(C) $\frac{51}{100}$
(D) $\frac{52}{101}$
(E) $\frac{13}{25}$

## Answer

**B.** $\frac{50}{99}$

## Solution

For $0\lt x\le100,$ the nearest lattice point above the line $y=\tfrac12x+2$ is $\left(x,\tfrac12x+3\right)$ if $x$ is even and $\left(x,\tfrac12x+\tfrac52\right)$ if $x$ is odd.

The slope from $(0,2)$ to that point is $\frac12+\frac1x$ for even $x$ and $\frac12+\frac{1}{2x}$ for odd $x.$ The minimum such slope is $\frac{51}{100}$ for even $x$ and $\frac{50}{99}$ for odd $x.$

Since $\frac{50}{99}\lt\frac{51}{100},$ the line avoids all these lattice points exactly when $\frac12\lt m\lt\frac{50}{99},$ so the maximum is $a=\frac{50}{99}.$

Thus, the correct answer is **B**.

## Key idea

A lattice point in the strip is determined by an integer slope; count the allowable reduced fractions under the given bounds.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
