# AMC12 2024-B Q06

![[assets/problems/2024-B/Q06.png]]

## Problem

The national debt is approximated by $5\cdot10^{13}$ dollars. How many digits does this number have when written in base $5$? Use $\log_{10}5\approx0.7$.

## Answer

- Correct choice: **(B) $20$**

## Solution

The number of base-$5$ digits of $N$ is

$$
\lfloor\log_5N\rfloor+1.
$$

For $N=5\cdot10^{13}$,

$$
\log_{10}N=13+\log_{10}5\approx13.7.
$$

Change the base:

$$
\log_5N=\frac{\log_{10}N}{\log_{10}5}
\approx\frac{13.7}{0.7}\approx19.57.
$$

Thus the number of digits is

$$
\lfloor19.57\rfloor+1=\boxed{20}.
$$

## Knowledge points

- Logarithm change of base
- Number of digits in a base

## Key idea

Digits in base $b$ are counted by $\lfloor\log_bN\rfloor+1$.

## Common mistakes

- Using $\lceil\log_5N\rceil$ without the endpoint check.
- Multiplying by $\log_{10}5$ instead of dividing.
