# AMC12 2020-A Q24

![[assets/problems/2020-A/Q24.png]]

## Problem

Suppose that $\triangle ABC$ is an equilateral triangle of side length $s$, with the property that there is a unique point $P$ inside the triangle such that $AP=1$, $BP=\sqrt3$, and $CP=2$. What is $s$?

## Answer

**B.** $\sqrt7$

## Solution

Place $A=(0,0)$, $B=(s,0)$, and $C=(s/2,\sqrt3s/2)$. Write $P=(x,y)$. From the differences of the squared distance equations,

$$x=\frac{s^2-2}{2s},\qquad y=\frac{s^2-4}{2\sqrt3s}.$$

Now use $AP^2=1$:

$$\left(\frac{s^2-2}{2s}\right)^2+\left(\frac{s^2-4}{2\sqrt3s}\right)^2=1.$$

Let $t=s^2$. After multiplying by $12s^2$ and simplifying,

$$t^2-8t+7=0,$$

so $t=1$ or $t=7$. A triangle of side $1$ cannot contain a point whose distance from $B$ is $\sqrt3$, so the interior condition selects $t=7$. Hence $s=\sqrt7$.

## Knowledge points

- Coordinate geometry
- Equilateral-triangle coordinates
- Solving distance equations

## Key idea

Subtract squared-distance equations to eliminate $x^2+y^2$ and then impose the remaining distance condition.

## Common mistakes

- Choosing $s=1$ without checking that $P$ is inside the triangle.
- Using the wrong height $\frac{\sqrt3s}{2}$ for vertex $C$.
