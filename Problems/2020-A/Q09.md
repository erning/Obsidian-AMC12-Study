# AMC12 2020-A Q09

![[assets/problems/2020-A/Q09.png]]

## Problem

How many solutions does the equation

$$\tan(2x)=\cos\left(\frac{x}{2}\right)$$

have on the interval $[0,2\pi]$?

## Answer

**E.** $5$

## Solution

The function $\cos(x/2)$ is bounded between $-1$ and $1$. On every branch of $\tan(2x)$, the tangent function is continuous and strictly increasing from $-\infty$ to $+\infty$.

The vertical asymptotes in $[0,2\pi]$ occur at

$$x=\frac\pi4,\frac{3\pi}4,\frac{5\pi}4,\frac{7\pi}4,$$

which split the interval into five branches. On each branch, tangent crosses the bounded cosine curve exactly once. Hence there are $5$ solutions.

## Knowledge points

- Period and asymptotes of tangent
- Intersection counting

## Key idea

Count branches, not explicit roots: each tangent branch sweeps through every real value while the cosine term remains bounded.

## Common mistakes

- Counting asymptotes as solutions.
- Forgetting that the interval contains five branches, not four.
