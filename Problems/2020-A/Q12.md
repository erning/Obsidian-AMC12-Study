# AMC12 2020-A Q12

![[assets/problems/2020-A/Q12.png]]

## Problem

Line $\ell$ in the coordinate plane has equation $3x-5y+40=0$. This line is rotated $45^\circ$ counterclockwise about $(20,20)$ to obtain line $k$. What is the $x$-coordinate of the $x$-intercept of line $k$?

## Answer

**B.** $15$

## Solution

![[assets/problems/2020-A/Q12-draft.svg]]

Rewrite $\ell$ as

$$y=\frac35x+8,$$

so its slope is $\frac35$. The point $(20,20)$ lies on $\ell$, because $60-100+40=0$. Therefore it remains on the rotated line $k$.

The slope after a $45^\circ$ counterclockwise rotation is

$$\frac{m+1}{1-m}=\frac{\frac35+1}{1-\frac35}=4.$$

Thus $k$ has equation $y-20=4(x-20)$. At the $x$-intercept, $y=0$, so

$$-20=4(x-20)\quad\Longrightarrow\quad x=15.$$

## Knowledge points

- Slope under rotation
- Point-slope form of a line

## Key idea

Rotate the direction vector through $45^\circ$ using the tangent addition formula, while retaining the fixed rotation center.

## Common mistakes

- Rotating the line about the origin.
- Using $\frac{m-1}{1+m}$, which corresponds to a different angle/sign convention.
