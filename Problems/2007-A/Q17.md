# AMC12 2007-A Q17

![[assets/problems/2007-A/Q17.png]]

## Problem

Suppose that $\sin a+\sin b=\sqrt{\tfrac53}$ and $\cos a+\cos b=1.$ What is $\cos(a-b)?$

(A) $\sqrt{\dfrac53}-1$
(B) $\dfrac13$
(C) $\dfrac12$
(D) $\dfrac23$
(E) $1$

## Answer

**B.** $\dfrac13$

## Solution

Squaring both equations gives $\sin^2 a+2\sin a\sin b+\sin^2 b=\tfrac53$ and $\cos^2 a+2\cos a\cos b$ $+\cos^2 b=1.$

Adding and using $\sin^2\theta+\cos^2\theta=1$ twice,
$$
 \begin{aligned} &2+2(\sin a\sin b+\cos a\cos b) \\ &=\tfrac83. \end{aligned}
$$

So $\cos(a-b)=\sin a\sin b$ $+\cos a\cos b$ $=\tfrac13.$

Thus, the correct answer is **B**.

## Key idea

Rewrite the expression using a standard identity before substituting values.
