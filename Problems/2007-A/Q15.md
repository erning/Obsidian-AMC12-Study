# AMC12 2007-A Q15

![[assets/problems/2007-A/Q15.png]]

## Problem

The set $\{3,6,9,10\}$ is augmented by a fifth element $n,$ not equal to any of the other four. The median of the resulting set is equal to its mean. What is the sum of all possible values of $n?$

(A) $7$
(B) $9$
(C) $19$
(D) $24$
(E) $26$

## Answer

**E.** $26$

## Solution

The mean is $\dfrac{28+n}{5}.$

If $n\lt 6,$ the median is $6,$ so $28+n=30$ and $n=2.$

If $6\lt n\lt 9,$ the median is $n,$ so $28+n=5n$ and $n=7.$

If $n\gt 9,$ the median is $9,$ so $28+n=45$ and $n=17.$

The sum of all possible values is $2+7+17=26.$

Thus, the correct answer is **E**.

## Key idea

Use the definition of the median together with the total sum from the mean.
