# AMC12 2019-B Q03

![[assets/problems/2019-B/Q03.png]]

## Problem

Which rigid transformation maps $\overline{AB}$ onto $\overline{A'B'}$ when $A(-2,1)$ maps to $A'(2,-1)$ and $B(-1,4)$ maps to $B'(1,-4)$?

## Answer

**E.** A $180^\circ$ rotation about the origin

## Solution

Both points follow the rule $(x,y)\mapsto(-x,-y)$:

$$(-2,1)\mapsto(2,-1),\qquad(-1,4)\mapsto(1,-4).$$

This rule is exactly a half-turn about the origin.

## Knowledge points

- Coordinate transformations
- Rotations about the origin

## Key idea

Identify a transformation rule that works for both endpoints.

## Common mistakes

- Confusing a half-turn with reflection in one coordinate axis.
- Verifying only one endpoint.
