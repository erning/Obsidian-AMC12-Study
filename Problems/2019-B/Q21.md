# AMC12 2019-B Q21

![[assets/problems/2019-B/Q21.png]]

## Problem

How many quadratic polynomials with real coefficients have their set of roots equal to their set of coefficients?

## Answer

**B.** $4$

## Solution

For $ax^2+bx+c$, the coefficient set has three entries while the root set has at most two values, so at least two coefficients must coincide. Applying Vieta’s relations

$$r+s=-\frac ba,\qquad rs=\frac ca$$

to the possible equality patterns gives the four polynomials

$$x^2+x-2,\quad -x^2-x,\quad x^2-x-\frac12,$$

and

$$ux^2+x+u,$$

where $u$ is the unique real root of $u^3+u+1=0$. They are distinct, so the count is 4.

## Knowledge points

- Vieta’s formulas
- Case analysis with sets

## Key idea

Use the fact that three coefficients must collapse to two distinct values.

## Common mistakes

- Treating coefficient and root multisets as identical.
- Forgetting the leading coefficient cannot be zero.
