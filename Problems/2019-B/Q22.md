# AMC12 2019-B Q22

![[assets/problems/2019-B/Q22.png]]

## Problem

Define $x_0=5$ and

$$x_{n+1}=\frac{x_n^2+5x_n+4}{x_n+6}.$$

Let $m$ be the least positive integer such that $x_m\le4+1/2^{20}$. Which interval contains $m$?

## Answer

**C.** $[81,242]$

## Solution

Set $a_n=x_n-4$. Since

$$x_n^2+5x_n+4=(x_n+6)(x_n-4)+20,$$

we obtain the more useful relation

$$a_{n+1}=a_n\frac{x_n+5}{x_n+6}.$$

The sequence decreases from $x_0=5$ toward 4, so

$$\frac9{10}<\frac{x_n+5}{x_n+6}<\frac{10}{11}.$$

Because $a_0=1$, this squeezes $a_m$ between $(9/10)^m$ and $(10/11)^m$. Comparing these bounds with $2^{-20}$ places the first possible $m$ between 81 and 242. Therefore $m\in[81,242]$.

## Knowledge points

- Recurrence transformations
- Exponential bounds

## Key idea

Subtract the limiting value 4 to expose a multiplicative recurrence.

## Common mistakes

- Assuming the recurrence is linear.
- Reversing the bounds because both ratios are below 1.
