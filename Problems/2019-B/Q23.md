# AMC12 2019-B Q23

![[assets/problems/2019-B/Q23.png]]

## Problem

How many binary sequences of length 19 begin and end with 0, contain no consecutive 0s, and contain no three consecutive 1s?

## Answer

**C.** $65$

## Solution

If there are $k$ zeros, the $k-1$ blocks of ones between the endpoint zeros each have length 1 or 2. The total number of ones is $19-k$. If $j$ blocks have length 2, then

$$j+(k-1)=19-k,$$

so $j=20-2k$. The constraints $0\le j\le k-1$ give $7\le k\le10$. For each $k$, choose the $j$ doubled blocks:

$$\binom66+\binom74+\binom82+\binom90=1+35+28+1=65.$$

Thus the answer is 65.

## Knowledge points

- Binary strings
- Stars-and-bars style block counting

## Key idea

Encode the restrictions as blocks of one or two 1s.

## Common mistakes

- Allowing a block of three 1s.
- Counting the endpoint zeros as movable.
