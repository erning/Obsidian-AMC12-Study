# AMC12 2019-B Q12

![[assets/problems/2019-B/Q12.png]]

## Problem

Right triangle $ACD$ is built outward on the hypotenuse $AC$ of isosceles right triangle $ABC$ with leg 1. The two triangles have equal perimeters. Find $\sin(2\angle BAD)$.

## Answer

**D.** $\frac79$

## Solution

![[assets/problems/2019-B/Q12-draft.svg]]

Since $AC=\sqrt2$, triangle $ABC$ has perimeter $2+\sqrt2$. Let $CD=d$; then $AD=\sqrt{2+d^2}$. Equal perimeters give

$$d+\sqrt{2+d^2}=2+\sqrt2-\sqrt2=2,$$

so squaring yields $d=1/2$ and $AD=3/2$. Let $\theta=\angle CAD$. Since $\angle BAC=45^\circ$, $2\angle BAD=90^\circ+2\theta$, hence $\sin(2\angle BAD)=\cos2\theta$. Now $\tan\theta=CD/AC=1/(2\sqrt2)$, so

$$\cos2\theta=\frac{1-1/8}{1+1/8}=\frac79.$$

## Knowledge points

- Right triangles
- Double-angle identities

## Key idea

Find $\tan\theta$ rather than the angle itself.

## Common mistakes

- Using $AC=1$ instead of $\sqrt2$.
- Forgetting the angle shift from $45^\circ$.
