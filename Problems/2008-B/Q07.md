# AMC12 2008-B Q07

![[assets/problems/2008-B/Q07.png]]

## Problem

For real numbers $a$ and $b,$ define $a \$ b = (a - b)^2.$ What is $(x - y)^2 \$ (y - x)^2?$

(A) $0$
(B) $x^2 + y^2$
(C) $2x^2$
(D) $2y^2$
(E) $4xy$

## Answer

**A.** $0$

## Solution

Since $(y - x)^2 = (x - y)^2,$ both inputs to the operation are the same value $t = (x - y)^2.$

Therefore $(x - y)^2 \$ (y - x)^2$ $= t \$ t$ $= (t - t)^2 = 0.$

Thus, the correct answer is **A**.

## Knowledge points

- Custom operation
- Symmetry

## Key idea

Recognize equal inputs or symmetry before expanding the custom operation.
