# AMC12 2008-B Q14

![[assets/problems/2008-B/Q14.png]]

## Problem

A circle has a radius of $\log_{10}(a^2)$ and a circumference of $\log_{10}(b^4).$ What is $\log_a b?$

(A) $\dfrac{1}{4\pi}$
(B) $\dfrac{1}{\pi}$
(C) $\pi$
(D) $2\pi$
(E) $10^{2\pi}$

## Answer

**C.** $\pi$

## Solution

The circumference is $2\pi$ times the radius, so
$$
 \log_{10}(b^4) = 2\pi \log_{10}(a^2).
$$

Rewriting, $4\log_{10} b = 4\pi \log_{10} a,$ hence $\log_{10} b = \pi \log_{10} a.$

Therefore $\log_a b = \dfrac{\log_{10} b}{\log_{10} a} = \pi.$

Thus, the correct answer is **C**.

## Knowledge points

- Logarithm
- Circumference

## Key idea

Apply the logarithm rules and rewrite the given quantity in a useful base.
