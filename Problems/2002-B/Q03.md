# AMC12 2002-B Q03

![[assets/problems/2002-B/Q03.png]]

## Problem

For how many positive integers $n$ is $n^2-3n+2$ a prime number?

(A) none
(B) one
(C) two
(D) more than two, but finitely many
(E) infinitely many

## Answer

**B.** one

## Solution

Factor $n^2-3n+2=(n-1)(n-2).$ For $n\ge4$ both factors exceed $1,$ so the value is composite. Checking $n=1,2,3$ gives $0,$ $0,$ and $2.$ Only $n=3$ yields a prime, so there is exactly one such $n.$

Thus, the correct answer is **B**.

## Key idea

- Factor $n^2-3n+2$
- $(n-1)(n-2)$ is a product of two integers; for it to be prime one factor must equal $1$
