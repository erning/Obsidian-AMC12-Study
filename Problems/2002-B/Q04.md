# AMC12 2002-B Q04

![[assets/problems/2002-B/Q04.png]]

## Problem

Let $n$ be a positive integer such that $\dfrac12+\dfrac13+\dfrac17+\dfrac1n$ is an integer. Which of the following statements is not true:

(A) $2$ divides $n$
(B) $3$ divides $n$
(C) $6$ divides $n$
(D) $7$ divides $n$
(E) $n\gt84$

## Answer

**E.** $n\gt84$

## Solution

Since $\dfrac12+\dfrac13+\dfrac17=\dfrac{41}{42},$ the sum $\dfrac{41}{42}+\dfrac1n$ lies strictly between $0$ and $2,$ so it must equal $1.$ Then $\dfrac1n=\dfrac1{42},$ giving $n=42.$

Now $2,$ $3,$ $6,$ and $7$ all divide $42,$ but $n\gt84$ is false. The untrue statement is $n\gt84.$

Thus, the correct answer is **E**.

## Key idea

- $\dfrac12+\dfrac13+\dfrac17=\dfrac{41}{42}$
- The whole sum lies strictly between $0$ and $2,$ so it must equal $1$; solve for $n$
