# AMC12 2002-B Q12

![[assets/problems/2002-B/Q12.png]]

## Problem

For how many integers $n$ is
$$
\dfrac{n}{20-n}
$$
 the square of an integer?

(A) $1$
(B) $2$
(C) $3$
(D) $4$
(E) $10$

## Answer

**D.** $4$

## Solution

Set $\dfrac{n}{20-n}=k^2.$ Solving, $n=\dfrac{20k^2}{k^2+1}.$ Since $k^2$ and $k^2+1$ are coprime, $k^2+1$ must divide $20,$ which happens only for $k=0,1,2,3.$

These give $n=0,$ $10,$ $16,$ $18,$ which is four values.

Thus, the correct answer is **D**.

## Key idea

- Set $\dfrac{n}{20-n}=k^2$ and solve for $n$ in terms of $k$
- $n=\dfrac{20k^2}{k^2+1}$; since $\gcd(k^2,k^2+1)=1,$ $k^2+1$ must divide $20$

- Divisibility

- Translate the wording into the correct equation, diagram, or counting model before calculating.
- Check units, endpoint conditions, and whether the requested quantity is a count, length, area, or probability.
