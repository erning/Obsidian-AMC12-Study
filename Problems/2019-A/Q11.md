# AMC12 2019-A Q11

![[assets/problems/2019-A/Q11.png]]

## Problem

For some positive integer $k$, the repeating base-$k$ representation of the base-ten fraction $\frac7{51}$ is

$$0.\overline{23}_k=0.232323\ldots_k.$$

What is $k$?

## Answer

**D.** $16$

## Solution

The repeating block $23_k$ has value $2k+3$. A two-digit repeating base-$k$ fraction therefore equals

$$0.\overline{23}_k=\frac{2k+3}{k^2-1}.$$

Set this equal to $7/51$:

$$51(2k+3)=7(k^2-1),$$

which simplifies to

$$7k^2-102k-160=0.$$

The positive root is

$$k=\frac{102+\sqrt{14884}}{14}=\frac{102+122}{14}=16.$$

## Knowledge points

- Repeating positional representations
- Geometric series
- Quadratic equations

## Key idea

Convert the repeating block into a geometric-series fraction in base $k$.

## Common mistakes

- Writing the denominator as $k^2$ instead of $k^2-1$.
- Forgetting that a base must be larger than the digit $3$.
