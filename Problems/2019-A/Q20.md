# AMC12 2019-A Q20

![[assets/problems/2019-A/Q20.png]]

## Problem

A random number in $[0,1]$ is chosen as follows. A fair coin is flipped. If it is heads, it is flipped again and the number is $0$ for heads and $1$ for tails. If the first flip is tails, the number is chosen uniformly from $[0,1]$. Two numbers $x$ and $y$ are chosen independently. What is the probability that $|x-y|>\frac12$?

## Answer

**B.** $\frac7{16}$

## Solution

For one random number,

$$P(x=0)=\frac14,\qquad P(x=1)=\frac14,$$

and the continuous uniform component has probability $1/2$.

Break the two-number experiment into types. The pairs $(0,1)$ and $(1,0)$ each contribute $1/16$. There are four point-uniform combinations; in each, the uniform value is more than $1/2$ away from the endpoint with conditional probability $1/2$, so each contributes $1/16$. Finally, the uniform-uniform combination has overall probability $1/4$, and two independent uniform numbers differ by more than $1/2$ with probability $1/4$, contributing another $1/16$.

Thus

$$P(|x-y|>1/2)=\frac{2+4+1}{16}=\frac7{16}.$$

## Knowledge points

- Total probability
- Mixed discrete-continuous distributions
- Uniform geometric probability

## Key idea

Separate the endpoint mass from the uniform component before combining the cases.

## Common mistakes

- Treating the distribution as uniform on $[0,1]$.
- Counting the endpoint cases but forgetting the four mixed cases.
