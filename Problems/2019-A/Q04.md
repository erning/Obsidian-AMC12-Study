# AMC12 2019-A Q04

![[assets/problems/2019-A/Q04.png]]

## Problem

What is the greatest number of consecutive integers whose sum is $45$?

## Answer

**D.** $90$

## Solution

Negative integers are allowed. The consecutive integers from $-44$ through $44$ sum to $0$ by pairing opposites. Adding $45$ gives the consecutive run

$$-44,-43,\ldots,44,45,$$

whose sum is $45$. Its length is

$$45-(-44)+1=90.$$

For a run of $L$ consecutive integers beginning at $u$, the sum is

$$\frac{L(2u+L-1)}2=45,$$

so $L$ must divide $90$. Since $90$ is the largest possible divisor not exceeding itself, no run longer than $90$ can work. Thus the maximum is $90$.

## Knowledge points

- Arithmetic sequences
- Symmetry of consecutive integers
- Bounds from averages

## Key idea

Center the sequence around $0$ so that the symmetric terms cancel.

## Common mistakes

- Assuming all terms must be positive.
- Checking only the standard positive decompositions of $45$.
