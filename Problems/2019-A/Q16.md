# AMC12 2019-A Q16

![[assets/problems/2019-A/Q16.png]]

## Problem

The numbers $1,2,\ldots,9$ are randomly placed into the squares of a $3\times3$ grid. What is the probability that the sum of the numbers in each row and each column is odd?

## Answer

**B.** $\frac1{14}$

## Solution

There are $5$ odd numbers and $4$ even numbers. A row or column has an odd sum exactly when it contains an odd number of odd entries.

With five odd entries total, the only possible parity pattern is a plus shape: one entire row and one entire column are occupied by odd entries. The row and column can be chosen in $3\cdot3=9$ ways; their intersection is counted only once, so this pattern has $3+3-1=5$ odd cells.

For each pattern, the $5$ odd numbers can be placed in $5!$ ways and the $4$ even numbers in $4!$ ways. Thus the probability is

$$\frac{9\cdot5!\cdot4!}{9!}=\frac1{14}.$$

## Knowledge points

- Parity
- Combinatorics
- Grid configurations

## Key idea

First count the possible locations of the odd numbers; the actual odd and even labels can then be permuted freely.

## Common mistakes

- Counting all arrangements of the nine numbers before imposing parity.
- Using a cross with $6$ cells instead of accounting for the shared center cell.
