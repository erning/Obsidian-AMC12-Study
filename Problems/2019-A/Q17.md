# AMC12 2019-A Q17

![[assets/problems/2019-A/Q17.png]]

## Problem

Let $s_k$ denote the sum of the $k$th powers of the roots of

$$x^3-5x^2+8x-13.$$

In particular, $s_0=3$, $s_1=5$, and $s_2=9$. Let $a,b,c$ be real numbers such that

$$s_{k+1}=as_k+bs_{k-1}+cs_{k-2}\qquad(k=2,3,\ldots).$$

What is $a+b+c$?

## Answer

**D.** $10$

## Solution

If $r$ is any root, then

$$r^3-5r^2+8r-13=0,$$

so

$$r^{k+1}=5r^k-8r^{k-1}+13r^{k-2}.$$

Summing this identity over all three roots gives

$$s_{k+1}=5s_k-8s_{k-1}+13s_{k-2}.$$

Therefore $a=5$, $b=-8$, and $c=13$, so

$$a+b+c=5-8+13=10.$$

## Knowledge points

- Polynomial roots
- Power-sum recurrence
- Vieta-style identities

## Key idea

Raise the defining polynomial equation for each root to the required power and sum.

## Common mistakes

- Using the coefficients with the wrong signs.
- Forgetting that summing over roots turns powers into the $s_k$ sequence.
