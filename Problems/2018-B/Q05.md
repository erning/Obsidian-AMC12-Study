# AMC12 2018-B Q05

![[assets/problems/2018-B/Q05.png]]

## Problem

How many subsets of $\{2,3,4,5,6,7,8,9\}$ contain at least one prime number?

## Answer

**D.** $240$

## Solution

There are $2^8=256$ total subsets. The prime-free subsets use only the four nonprimes $4,6,8,9$, so there are $2^4=16$ of them. Thus the desired count is

$$256-16=240.$$

## Knowledge points

- Complementary counting
- Subsets

## Key idea

Count subsets with no prime, then subtract from all subsets.

## Common mistakes

- Treating 1 as an element of the set.
- Counting subsets containing exactly one prime.
