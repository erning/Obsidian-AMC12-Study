# AMC12 2018-B Q24

![[assets/problems/2018-B/Q24.png]]

## Problem

Let $\lfloor x\rfloor$ be the greatest integer at most $x$. How many real numbers satisfy

$$x^2+10{,}000\lfloor x\rfloor=10{,}000x?$$

## Answer

**C.** $199$

## Solution

Write $x=\lfloor x\rfloor+\{x\}$. The equation becomes

$$x^2=10{,}000\{x\}.$$

Since $0\le\{x\}<1$, we need $-100<x<100$. On each interval $[k,k+1)$, write $x=k+t$ with $0\le t<1$. The equation is

$$ (k+t)^2-10{,}000t=0.$$

The left side is strictly decreasing over the relevant interval and crosses zero exactly once for each $k=-100,-99,\ldots,98$. There are $199$ such integers $k$.

## Knowledge points

- Floor and fractional-part functions
- Interval counting

## Key idea

Reduce the equation to one solution on each unit interval.

## Common mistakes

- Including $x=\pm100$, which violates the fractional-part bound.
