# AMC12 2018-B Q15

![[assets/problems/2018-B/Q15.png]]

## Problem

How many 3-digit positive odd multiples of 3 do not contain the digit 3?

## Answer

**A.** $96$

## Solution

The hundreds digit has 8 choices: $1,2,4,5,6,7,8,9$. The units digit must be odd and cannot be 3, giving 4 choices: $1,5,7,9$. The tens digit can be any of the nine digits other than 3. These split into three residue classes modulo 3, each with three choices, and exactly one class makes the digit sum divisible by 3. Therefore

$$8\cdot4\cdot3=96.$$

## Knowledge points

- Divisibility by 3
- Digit counting

## Key idea

Use the digit-sum test and separate the hundreds, tens, and units choices.

## Common mistakes

- Allowing 3 in one of the digits.
- Treating 0 as a possible hundreds digit.
