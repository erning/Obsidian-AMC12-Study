# AMC12 2018-B Q09

![[assets/problems/2018-B/Q09.png]]

## Problem

Evaluate

$$\sum_{i=1}^{100}\sum_{j=1}^{100}(i+j).$$

## Answer

**E.** $1{,}010{,}000$

## Solution

Split the sum:

$$\sum_{i,j}(i+j)=100\sum_{i=1}^{100}i+100\sum_{j=1}^{100}j.$$

Since $\sum_{k=1}^{100}k=5050$, the value is

$$100(5050)+100(5050)=1{,}010{,}000.$$

## Knowledge points

- Double sums
- Arithmetic series

## Key idea

Each fixed $i$ or $j$ is repeated 100 times.

## Common mistakes

- Forgetting one of the two multiplicity factors.
