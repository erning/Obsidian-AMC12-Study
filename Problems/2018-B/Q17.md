# AMC12 2018-B Q17

![[assets/problems/2018-B/Q17.png]]

## Problem

Positive integers $p,q$ satisfy

$$\frac59<\frac pq<\frac47,$$

with $q$ as small as possible. Find $q-p$.

## Answer

**A.** $7$

## Solution

The inequalities imply the positive integers

$$9p-5q\ge1,\qquad4q-7p\ge1.$$

Write

$$\frac47-\frac59=\frac{16}{63}=\frac{4q-7p}{7q}+\frac{9p-5q}{9q}\ge\frac1{7q}+\frac1{9q}=\frac{16}{63q}.$$

Thus $q\ge16$. With $q=16$, $p=9$ satisfies $5/9<9/16<4/7$. Therefore $q-p=7$.

## Knowledge points

- Rational approximation
- Farey-style inequalities

## Key idea

Express the gap between the endpoints using the two positive integer errors.

## Common mistakes

- Checking only a fraction with small numerator.
- Forgetting both inequalities must be strict.
