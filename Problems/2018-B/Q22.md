# AMC12 2018-B Q22

![[assets/problems/2018-B/Q22.png]]

## Problem

How many degree-at-most-3 polynomials with coefficients in $\{0,1,\ldots,9\}$ satisfy $P(-1)=-9$?

## Answer

**D.** $220$

## Solution

Write $P(x)=ax^3+bx^2+cx+d$. The condition is

$$-a+b-c+d=-9.$$

Set $a'=9-a$ and $c'=9-c$. Then

$$a'+b+c'+d=9,$$

with all four variables nonnegative. Stars and bars gives

$$\binom{9+4-1}{4-1}=\binom{12}{3}=220.$$

The upper bounds are automatic because the sum is only 9.

## Knowledge points

- Coefficient equations
- Stars and bars

## Key idea

Complement the alternating coefficients to turn the equation into a nonnegative sum.

## Common mistakes

- Counting degree exactly 3 only.
- Forgetting that zero coefficients are allowed.
