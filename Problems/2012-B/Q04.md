# AMC12 2012-B Q04

![[assets/problems/2012-B/Q04.png]]

## Problem

Suppose that the euro is worth $1.30$ dollars. If Diana has $500$ dollars and Étienne has $400$ euros, by what percent is the value of Étienne's money greater than the value of Diana's money?

(A) $2$
(B) $4$
(C) $6.5$
(D) $8$
(E) $13$

## Answer

**B.** $4$

## Solution

Étienne's money is worth $400\cdot1.30=520$ dollars, while Diana has $500$ dollars.

The percent by which Étienne's value exceeds Diana's is $$\frac{520-500}{500}\cdot100\%=4\%.$$

Thus, the correct answer is **B**.

## Knowledge points

- Percentage
- Unit conversion

## Key idea

Convert both amounts to the same currency before calculating the percent increase relative to Diana’s amount.

## Common mistakes

- Using a quantity in the wrong units or applying a formula before identifying the relevant structure.
- Losing a boundary condition, an excluded case, or a restriction from the original statement.
