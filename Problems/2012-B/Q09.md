# AMC12 2012-B Q09

![[assets/problems/2012-B/Q09.png]]

## Problem

It takes Clea $60$ seconds to walk down an escalator when it is not operating, and only $24$ seconds to walk down the escalator when it is operating. How many seconds does it take Clea to ride down the operating escalator when she just stands on it?

(A) $36$
(B) $40$
(C) $42$
(D) $48$
(E) $52$

## Answer

**B.** $40$

## Solution

Let $x$ be Clea's walking rate and $r$ the escalator's rate, with the length equal to $60x.$ Walking on the moving escalator gives $24(x+r)=60x,$ so $r=\tfrac32x.$

Standing takes time $t$ with $rt=60x,$ so $\tfrac32xt=60x$ and $t=40$ seconds.

Thus, the correct answer is **B**.

## Knowledge points

- Rate
- System of equations

## Key idea

Write the escalator length using Clea’s walking rate, then compare the combined rate with the moving-escalator time.
