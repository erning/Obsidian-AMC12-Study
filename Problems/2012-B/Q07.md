# AMC12 2012-B Q07

![[assets/problems/2012-B/Q07.png]]

## Problem

Small lights are hung on a string $6$ inches apart in the order red, red, green, green, green, red, red, green, green, green, and so on continuing this pattern of $2$ red lights followed by $3$ green lights. How many feet separate the $3$rd red light and the $21$st red light?

Note: $1$ foot is equal to $12$ inches.

(A) $18$
(B) $18.5$
(C) $20$
(D) $20.5$
(E) $22.5$

## Answer

**E.** $22.5$

## Solution

The lights repeat in blocks of $5,$ so consecutive blocks start $5\cdot6=30$ inches, or $2.5$ feet, apart.

Each block has one odd-numbered red light beginning it. The $3$rd red light begins the $2$nd block and the $21$st red light begins the $11$th block.

The distance between them is $(11-2)\cdot2.5=22.5$ feet.

Thus, the correct answer is **E**.

## Knowledge points

- Pattern recognition
- Unit conversion

## Key idea

Red lights repeat every five positions, so compare the positions of the third and twenty-first red lights and convert the distance to feet.

## Common mistakes

- Using a quantity in the wrong units or applying a formula before identifying the relevant structure.
- Losing a boundary condition, an excluded case, or a restriction from the original statement.
