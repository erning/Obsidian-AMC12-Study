# AMC12 2023-B Q19

![[assets/problems/2023-B/Q19.png]]

## Problem

Each of $2023$ balls is randomly placed into one of $3$ bins. Which answer is closest to the probability that each bin contains an odd number of balls?

## Answer

- Correct choice: **(E) $\frac14$**

## Solution

Use the parity filter. For an odd number $n$, the number of assignments in which all three bin counts are odd is
$$
\frac18\left(3^n-3\cdot1^n+3(-1)^n-(-3)^n\right)
=\frac{3^n-3}{4}.
$$
There are $3^n$ total assignments, so the probability is
$$
\frac{3^n-3}{4\cdot3^n}
=\frac14-\frac1{4\cdot3^{n-1}}.
$$
For $n=2023$, this is extremely close to $\frac14$, so the closest choice is
$$
\boxed{\frac14}.
$$

## Knowledge points

- Basic probability
- Parity
- Inclusion-exclusion filters

## Key idea

Use signs to filter assignments by the parity of each bin count.

## Common mistakes

- Treating the three bin counts as independent.
