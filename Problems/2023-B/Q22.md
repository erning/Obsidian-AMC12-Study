# AMC12 2023-B Q22

![[assets/problems/2023-B/Q22.png]]

## Problem

A real-valued function $f$ satisfies, for all real numbers $a,b$,
$$
f(a+b)+f(a-b)=2f(a)f(b).
$$
Which of the following cannot be the value of $f(1)$?

## Answer

- Correct choice: **(E) $-2$**

## Solution

Set $a=b=0$:
$$
2f(0)=2f(0)^2,
$$
so $f(0)=0$ or $f(0)=1$. If $f(0)=0$, setting $b=0$ forces $f\equiv0$, so $f(1)=0$.

Otherwise $f(0)=1$. Setting $a=b$ gives
$$
f(2a)+f(0)=2f(a)^2,
$$
so
$$
f(2a)=2f(a)^2-1\ge-1.
$$
Taking $a=\frac12$ shows $f(1)\ge-1$. Therefore $f(1)=-2$ is impossible. The other listed values can occur, so the answer is
$$
\boxed{-2}.
$$

## Knowledge points

- Bounding

## Key idea

Use $a=b=0$ and then $a=b$ to obtain a lower bound for $f(1)$.

## Common mistakes

- Assuming $f(0)=1$ without handling the $f\equiv0$ case.
