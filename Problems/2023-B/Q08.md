# AMC12 2023-B Q08

![[assets/problems/2023-B/Q08.png]]

## Problem

How many nonempty subsets $B$ of $\{0,1,2,3,\ldots,12\}$ have the property that the number of elements in $B$ equals the least element of $B$?

## Answer

- Correct choice: **(D) $144$**

## Solution

If the least element is $m$, then $m\ge1$ because the set is nonempty and its size is positive. The set must contain $m$ and choose its other $m-1$ elements from $\{m+1,\ldots,12\}$, which has $12-m$ elements. Thus the count is
$$
\sum_{m=1}^{6}\binom{12-m}{m-1}
=\binom{11}{0}+\binom{10}{1}+\binom{9}{2}
+\binom{8}{3}+\binom{7}{4}+\binom{6}{5}.
$$
This equals
$$
1+10+36+56+35+6=\boxed{144}.
$$

## Knowledge points

- Combinations
- Least-element casework

## Key idea

Fix the least element, then choose the remaining elements above it.

## Common mistakes

- Allowing $m=0$, even though a set with least element $0$ cannot have size $0$.
