# AMC12 2010-A Q04

![[assets/problems/2010-A/Q04.png]]

## Problem

If $x\lt0,$ then which of the following must be positive?

(A) $\frac{x}{|x|}$
(B) $-x^2$
(C) $-2^x$
(D) $-x^{-1}$
(E) $\sqrt[3]{x}$

## Answer

**D.** $-x^{-1}$

## Solution

Choice (D) is $-x^{-1}=-\frac1x.$ When $x\lt0,$ $\frac1x\lt0,$ so $-\frac1x\gt0.$

Testing $x=-1$ shows the other choices need not be positive: $\frac{x}{|x|}=-1,$ $-x^2=-1,$ $-2^x=-\tfrac12,$ and $\sqrt[3]{x}=-1.$

Thus, **D** is the correct answer.

## Knowledge points

- Inequality
- Exponent
- Absolute value

## Key idea

Check the sign of each expression when $x<0$; the cube root preserves the sign while the other choices do not always become positive.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
