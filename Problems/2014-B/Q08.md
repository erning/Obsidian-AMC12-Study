# AMC12 2014-B Q08

![[assets/problems/2014-B/Q08.png]]

## Problem

In the displayed addition, $A,B,C,D$ are distinct digits. How many different values are possible for $D$?

## Answer

**C.** $7$

## Solution

The leftmost column has no outgoing carry, so
$$A+B=D,\qquad A+B\le9.$$
The tens and thousands columns force $C=0$ and eliminate carries. Thus $D=A+B$, where $A$ and $B$ are distinct positive digits. The possible sums run from $3$ through $9$, giving
$$9-3+1=7$$
values.

## Knowledge points

- Column addition
- Carry analysis
- Digit constraints

## Key idea

Use the columns that must produce the same digit to eliminate carries first.

## Common mistakes

- Allowing $A$ or $B$ to be zero
- Counting digit pairs instead of possible sums
