# AMC12 2014-B Q01

![[assets/problems/2014-B/Q01.png]]

## Problem

Leah has $13$ coins, all pennies and nickels. If she had one more nickel than she has now, she would have the same number of pennies and nickels. What are her coins worth in cents?

## Answer

**C.** $37$

## Solution

Let $n$ be the number of nickels. Then there are $13-n$ pennies. The condition gives
$$n+1=13-n,$$
so $n=6$. There are $6$ nickels and $7$ pennies, worth
$$6\cdot5+7=37\text{ cents}.$$

## Knowledge points

- Linear equations
- Counting coins
- Place value

## Key idea

Express the number of pennies in terms of the number of nickels.

## Common mistakes

- Confusing the current and hypothetical coin counts
- Using $5$ cents for every coin
