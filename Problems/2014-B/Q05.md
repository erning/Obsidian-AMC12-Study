# AMC12 2014-B Q05

![[assets/problems/2014-B/Q05.png]]

## Problem

Doug constructs a square window from $8$ equal-size panes. Each pane's height-to-width ratio is $5:2$, and borders around and between panes are $2$ inches wide. What is the side length of the window?

## Answer

**A.** $26$

## Solution

Let a pane have width $2x$ and height $5x$. The window has four panes across and two down. Its width is
$$4(2x)+5(2)=8x+10,$$
and its height is
$$2(5x)+3(2)=10x+6.$$
Equating them gives $x=2$, so the side length is
$$10(2)+6=26.$$

## Knowledge points

- Ratios
- Linear equations
- Geometric dimensions

## Key idea

Compute the two overall dimensions including every border.

## Common mistakes

- Counting only the outer borders
- Reversing the $5:2$ ratio
