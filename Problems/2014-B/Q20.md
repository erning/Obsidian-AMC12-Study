# AMC12 2014-B Q20

![[assets/problems/2014-B/Q20.png]]

## Problem

For how many positive integers $x$ is
$$\log_{10}(x-40)+\log_{10}(60-x)<2?$$

## Answer

**B.** $18$

## Solution

The logarithms require
$$40<x<60.$$
Combining them gives
$$\log_{10}\bigl((x-40)(60-x)\bigr)<2,$$
so
$$(x-40)(60-x)<100.$$
This is equivalent to
$$(x-50)^2>0.$$
Thus every integer from $41$ through $59$ works except $x=50$, giving
$$19-1=18.$$

## Knowledge points

- Logarithmic inequalities
- Domain restrictions
- Quadratic inequalities

## Key idea

Apply the logarithm product rule only after enforcing both domains.

## Common mistakes

- Including $x=40$ or $x=60$
- Treating the strict inequality as non-strict
