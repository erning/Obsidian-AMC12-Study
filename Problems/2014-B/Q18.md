# AMC12 2014-B Q18

![[assets/problems/2014-B/Q18.png]]

## Problem

The numbers $1,2,3,4,5$ are arranged in a circle. An arrangement is bad if it is not true that for every $n$ from $1$ to $15$ there is a consecutive subset summing to $n$. Arrangements equivalent under rotation or reflection are identified. How many bad arrangements are there?

## Answer

**B.** $2$

## Solution

A single number covers sums $1$ through $5$. If a consecutive block sums to $n$, the complementary block sums to $15-n$, so many sums are automatic. An arrangement can fail only to produce $6$ or $7$.
Checking the cyclic orders up to rotation and reflection gives exactly two bad arrangements:
$$14352\quad\text{and}\quad23154.$$
Thus there are $2$ bad arrangements.

## Knowledge points

- Circular arrangements
- Consecutive sums
- Symmetry reduction

## Key idea

Use complementary blocks to reduce the sums that must be checked.

## Common mistakes

- Counting rotations as distinct
- Checking arbitrary subsets instead of consecutive blocks
