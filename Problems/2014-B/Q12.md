# AMC12 2014-B Q12

![[assets/problems/2014-B/Q12.png]]

## Problem

A set $S$ consists of triangles with integer side lengths less than $5$, with no two triangles congruent or similar. What is the greatest possible size of $S$?

## Answer

**B.** $9$

## Solution

Up to order, the valid integer triangles are
$$
(2,2,1),(3,2,2),(3,3,1),(3,3,2),(4,2,3),
(4,3,2),(4,3,3),(4,4,1),(4,4,3),(4,4,2).
$$
The equilateral triangle contributes one. The similar pair $(2,2,1)$ and $(4,4,2)$ contributes only one. The remaining seven are pairwise non-similar, so the maximum is
$$1+1+7=9.$$

## Knowledge points

- Triangle inequalities
- Similarity classes
- Systematic enumeration

## Key idea

List integer triangles first, then select one representative from each similarity class.

## Common mistakes

- Counting different scalings as different choices
- Including degenerate triples
