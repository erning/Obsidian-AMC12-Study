# AMC12 2014-B Q21

![[assets/problems/2014-B/Q21.png]]

## Problem

In the figure, $ABCD$ is a square of side length $1$. The rectangles $JKHG$ and $EBCF$ are congruent. What is $BE$?

## Answer

**C.** $2-\sqrt3$

## Solution

Let $x=BE=GH=CF$ and let $\theta$ be the indicated acute angle. In the right triangle,
$$x=\frac{\sin\theta}{1-\cos\theta}.$$
Using the side decomposition of the square gives
$$1=x+\sin\theta+x\cos\theta.$$
Substitute the expression for $x$:
$$1=2\sin\theta,$$
so $\theta=30^\circ$. Therefore
$$x=\frac{1/2}{1-\sqrt3/2}=2-\sqrt3.$$

## Knowledge points

- Similar triangles
- Trigonometric identities
- Geometric decomposition

## Key idea

Express the congruent rectangle dimensions using one angle, then use the unit side.

## Common mistakes

- Using the complementary angle
- Losing the factor $x\cos\theta$ in the side sum
