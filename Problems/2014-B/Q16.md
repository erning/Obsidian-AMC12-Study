# AMC12 2014-B Q16

![[assets/problems/2014-B/Q16.png]]

## Problem

Let $P$ be a cubic polynomial with $P(0)=k$, $P(1)=2k$, and $P(-1)=3k$. What is $P(2)+P(-2)$?

## Answer

**E.** $14k$

## Solution

Write
$$P(x)=ax^3+bx^2+cx+k.$$
The conditions at $1$ and $-1$ give
$$a+b+c+k=2k,\qquad-a+b-c+k=3k.$$
Adding yields $2b+2k=5k$, so $2b=3k$. The odd powers cancel in
$$P(2)+P(-2)=8b+2k=4(2b)+2k=14k.$$

## Knowledge points

- Polynomial coefficients
- Symmetry of $P(x)+P(-x)$

## Key idea

Add the values at opposite inputs so the odd powers cancel.

## Common mistakes

- Solving for every coefficient unnecessarily
- Forgetting the constant term appears twice
