# AMC12 2018-A Q13

![[assets/problems/2018-A/Q13.png]]

## Problem

How many nonnegative integers can be written as

$$\sum_{i=0}^{7}a_i3^i,$$

where each $a_i\in\{-1,0,1\}$?

## Answer

**D.** $3281$

## Solution

Replace each $a_i$ by $a_i+1$. This changes the expression by the fixed amount

$$\sum_{i=0}^{7}3^i,$$

and gives all base-3 numerals with eight digits, so there are $3^8=6561$ distinct values. The original values are symmetric about 0 because negating all coefficients negates the sum. Thus one value is 0 and half of the remaining values are positive. The number of nonnegative values is

$$1+\frac{6561-1}{2}=3281.$$

## Knowledge points

- Balanced ternary
- Symmetry of finite sets

## Key idea

Shift the coefficients to ordinary base-3 digits.

## Common mistakes

- Counting $3^8$ as the number of nonnegative values.
- Forgetting the central value 0 when halving.
