# AMC12 2018-A Q05

![[assets/problems/2018-A/Q05.png]]

## Problem

Find the sum of all possible $k$ for which $x^2-3x+2$ and $x^2-5x+k$ have a common root.

## Answer

**E.** $10$

## Solution

Factor the first polynomial:

$$x^2-3x+2=(x-1)(x-2).$$

If the shared root is 1, then $1-5+k=0$, so $k=4$. If it is 2, then $4-10+k=0$, so $k=6$. These are the only possibilities, and their sum is $4+6=10$.

## Knowledge points

- Factoring quadratics
- Common roots

## Key idea

Test each root of the already factored polynomial.

## Common mistakes

- Looking for a root common to the coefficients instead of the polynomials.
- Forgetting that the first polynomial has two roots.
