# AMC12 2018-A Q12

![[assets/problems/2018-A/Q12.png]]

## Problem

Let $S$ be a 6-element subset of $\{1,2,\ldots,12\}$ such that if $a<b$ are in $S$, then $b$ is not a multiple of $a$. What is the least possible element of $S$?

## Answer

**C.** $4$

## Solution

Partition the numbers into divisibility chains:

$$\{1,2,4,8\},\ \{3,6,12\},\ \{5,10\},\ \{7\},\ \{9\},\ \{11\}.$$

At most one number can be chosen from each chain, and six elements force one from every chain. Therefore $7,9,11$ are selected. Since 9 is selected, neither 1 nor 3 can be selected; the second chain contributes 6 or 12, and the first chain cannot contribute 1 or 2. Choosing 4 works, for example

$$S=\{4,5,6,7,9,11\}.$$

Hence the least possible element is 4.

## Knowledge points

- Divisibility chains
- Extremal set construction

## Key idea

Use chains to obtain an upper bound, then construct a set attaining it.

## Common mistakes

- Checking only pairwise numerical distance rather than divisibility.
- Giving a lower bound without constructing a valid set.
