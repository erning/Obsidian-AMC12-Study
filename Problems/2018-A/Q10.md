# AMC12 2018-A Q10

![[assets/problems/2018-A/Q10.png]]

## Problem

How many ordered pairs of real numbers satisfy

$$x+3y=3,\qquad |x|-|y|=1?$$

## Answer

**C.** $3$

## Solution

The absolute-value equation means either $|x|=|y|+1$ or $|x|=|y|-1$. Equivalently, test the four lines $x=y+1$, $x=y-1$, $x=-y+1$, and $x=-y-1$ against $x+3y=3$. They produce

$$\left(\frac32,\frac12\right),\quad(0,1),\quad(0,1),\quad(-3,2).$$

The duplicate $(0,1)$ is one solution, so there are three distinct ordered pairs.

## Knowledge points

- Absolute values
- Piecewise linear systems

## Key idea

Replace the absolute values by the four possible sign combinations and remove duplicates.

## Common mistakes

- Counting the duplicate solution twice.
- Using only $x=\pm y+1$ without checking all signs.
