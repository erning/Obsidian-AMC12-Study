# AMC12 2018-A Q01

![[assets/problems/2018-A/Q01.png]]

## Problem

A large urn contains 100 balls, 36% red and the rest blue. How many blue balls must be removed so that red balls are 72% of the urn?

## Answer

**D.** $50$

## Solution

There are $0.36(100)=36$ red balls, and none are removed. If the final total is $T$, then

$$\frac{36}{T}=0.72,$$

so $T=50$. The number of blue balls removed is $100-50=50$.

## Knowledge points

- Percentages
- Equations with a fixed numerator

## Key idea

Find the final total from the unchanged number of red balls.

## Common mistakes

- Removing red balls in the calculation.
- Treating 72% as the fraction of blue balls.
