# AMC12 2018-A Q07

![[assets/problems/2018-A/Q07.png]]

## Problem

For how many integer values of $n$ is

$$4000\left(\frac25\right)^n$$

an integer?

## Answer

**E.** $9$

## Solution

Since $4000=2^5\cdot5^3$,

$$4000\left(\frac25\right)^n=2^{5+n}5^{3-n}.$$

This is an integer exactly when both exponents are nonnegative:

$$5+n\ge0,\qquad3-n\ge0.$$

Thus $-5\le n\le3$, which contains $3-(-5)+1=9$ integers.

## Knowledge points

- Prime exponents
- Integer powers

## Key idea

Track the exponents of 2 and 5 separately.

## Common mistakes

- Restricting $n$ to positive integers when the problem does not.
- Requiring the exponents to be positive rather than nonnegative.
