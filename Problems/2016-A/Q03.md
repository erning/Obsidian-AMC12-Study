# AMC12 2016-A Q03

![[assets/problems/2016-A/Q03.png]]

## Problem

For $y\ne0$, define $\operatorname{rem}(x,y)=x-y\left\lfloor\frac{x}{y}\right\rfloor$. Find $\operatorname{rem}\left(\frac38,-\frac25\right)$.

## Answer

**B.** $-\frac1{40}$

## Solution

First,
$$\frac{x}{y}=\frac{3/8}{-2/5}=-\frac{15}{16},\qquad\left\lfloor-\frac{15}{16}\right\rfloor=-1.$$
Therefore
$$\operatorname{rem}\left(\frac38,-\frac25\right)=\frac38-\left(-\frac25\right)(-1)=\frac38-\frac25=-\frac1{40}.$$

## Knowledge points

- Floor function
- Remainder with a negative divisor

## Key idea

Evaluate the floor of the quotient carefully; it is $-1$, not $0$.

## Common mistakes

- Rounding $-15/16$ toward zero
- Forgetting the second minus sign
