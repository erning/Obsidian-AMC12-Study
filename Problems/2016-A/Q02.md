# AMC12 2016-A Q02

![[assets/problems/2016-A/Q02.png]]

## Problem

Find $x$ if $10^x\cdot100^{2x}=1000^5$.

## Answer

**C.** $3$

## Solution

Use $100=10^2$ and $1000=10^3$:
$$10^x\cdot10^{4x}=10^{15}.$$
Thus $5x=15$, so $x=3$.

## Knowledge points

- Exponent laws
- Converting all quantities to one base

## Key idea

Rewrite every power as a power of $10$.

## Common mistakes

- Reading $100^{2x}$ as $10^{2x}$
- Equating exponents before using a common base
