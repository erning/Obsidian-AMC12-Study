# AMC12 2016-A Q13

![[assets/problems/2016-A/Q13.png]]

## Problem

For a positive multiple $N$ of $5$, a red ball and $N$ green balls are arranged in a line. Find the digit sum of the least $N$ with $P(N)<321/400$.

## Answer

**A.** $12$

## Solution

Write $N=5k$. The red ball has $5k+1$ possible positions. The positions for which fewer than $3/5$ of the green balls lie on each side number $k-1$, so
$$P(N)=1-\frac{k-1}{5k+1}=\frac{4k+2}{5k+1}.$$
Solving
$$\frac{4k+2}{5k+1}<\frac{321}{400}$$
gives $k>95.8$, hence $k=96$, $N=480$. The digit sum is $12$.

## Knowledge points

- Counting positions
- Inequalities with rational expressions

## Key idea

Parameterize the required multiple as $N=5k$ and count red-ball positions.

## Common mistakes

- Counting green-ball arrangements instead of red positions
- Reversing the probability inequality
