# AMC12 2016-A Q20

![[assets/problems/2016-A/Q20.png]]

## Problem

A binary operation satisfies $a\diamond(b\diamond c)=(a\diamond b)c$ and $a\diamond a=1$. Solve $2016\diamond(6\diamond x)=100$ and find $p+q$ if $x=p/q$.

## Answer

**A.** $109$

## Solution

First, $a\diamond1=a$. Then
$$a=a\diamond1=a\diamond(b\diamond b)=(a\diamond b)b,$$
so
$$a\diamond b=\frac ab.$$
Therefore
$$2016\diamond(6\diamond x)=\frac{2016}{6/x}=336x=100,$$
so $x=25/84$ and $p+q=109$.

## Knowledge points

- Functional identities
- Deriving an operation from axioms

## Key idea

Use $a\diamond a=1$ to derive the identity element and then the quotient rule.

## Common mistakes

- Assuming the operation is ordinary division without proof
- Reversing the order in $a/b$
