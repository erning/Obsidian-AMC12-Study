# AMC12 2016-A Q14

![[assets/problems/2016-A/Q14.png]]

## Problem

Label cube vertices with $1$ through $8$ so every face has the same sum; identify arrangements under rotations. How many?

## Answer

**C.** $6$

## Solution

Each vertex belongs to three faces. If the common face sum is $S$, then
$$6S=3(1+2+\cdots+8)=108,$$
so $S=18$. The possible face containing $1$ forces $1$ and $8$ to be adjacent. Fix their adjacent positions by rotation; the remaining three labels on that face can be arranged in $3!=6$ ways, and the opposite labels are forced.

## Knowledge points

- Invariant counting
- Symmetry under rotations

## Key idea

Find the face sum globally before fixing a rotation.

## Common mistakes

- Counting reflected arrangements separately
- Assuming any four labels summing to $18$ can be a face
