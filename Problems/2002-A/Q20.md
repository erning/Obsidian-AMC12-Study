# AMC12 2002-A Q20

![[assets/problems/2002-A/Q20.png]]

## Problem

Suppose that $a$ and $b$ are digits, not both nine and not both zero, and the repeating decimal $0.\overline{ab}$ is expressed as a fraction in lowest terms. How many different denominators are possible?

(A) $3$
(B) $4$
(C) $5$
(D) $8$
(E) $9$

## Answer

**C.** $5$

## Solution

Since $0.\overline{ab} = \dfrac{\overline{ab}}{99},$ the reduced denominator divides $99 = 3^2\cdot 11.$ The divisors are $1, 3, 9, 11, 33, 99.$

The denominator $1$ would require $\overline{ab} = 99,$ i.e. $a = b = 9,$ which is excluded. Each of $3, 9, 11, 33, 99$ is achievable, giving $5$ possible denominators.

Thus, the correct answer is **C**.

## Key idea

- $0.\overline{ab} = \dfrac{\overline{ab}}{99}$ where $\overline{ab}$ is the two-digit value
- In lowest terms the denominator is a divisor of $99 = 3^2\cdot 11;$ exclude $1$ since $a, b$ are not both $9$
