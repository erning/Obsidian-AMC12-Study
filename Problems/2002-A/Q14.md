# AMC12 2002-A Q14

![[assets/problems/2002-A/Q14.png]]

## Problem

For all positive integers $n,$ let $f(n) = \log_{2002} n^2.$ Let
$$
N = f(11) + f(13) + f(14).
$$
 Which of the following relations is true?

(A) $N \gt 1$
(B) $N = 1$
(C) $1 \lt N \lt 2$
(D) $N = 2$
(E) $N \gt 2$

## Answer

**D.** $N = 2$

## Solution

Using $\log a^2 = 2\log a$ and adding logs,
$$
 \begin{aligned} N &= \log_{2002} 11^2 + \log_{2002} 13^2 \\ &\quad {}+ \log_{2002} 14^2 \\ &= \log_{2002}(11\cdot 13\cdot 14)^2. \end{aligned}
$$

Since $11\cdot 13\cdot 14 = 2002,$ this is $\log_{2002} 2002^2 = 2.$

Thus, the correct answer is **D**.

## Key idea

- $f(n) = \log_{2002} n^2 = 2\log_{2002} n$
- Combine into $N = \log_{2002}(11\cdot 13\cdot 14)^2$ and evaluate the product inside
