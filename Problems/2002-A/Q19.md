# AMC12 2002-A Q19

![[assets/problems/2002-A/Q19.png]]

## Problem

The graph of the function $f$ is shown below. How many solutions does the equation $f(f(x)) = 6$ have?

(A) $2$
(B) $4$
(C) $5$
(D) $6$
(E) $7$

## Answer

**D.** $6$

## Solution

The graph reaches $6$ at $x = -2$ and $x = 1,$ so $f(f(x)) = 6$ requires $f(x) = -2$ or $f(x) = 1.$

The horizontal line $y = -2$ meets the graph twice, and $y = 1$ meets it four times, giving $2 + 4 = 6$ solutions.

Thus, the correct answer is **D**.

## Key idea

- $f(f(x)) = 6$ means $f(x)$ must be an input where $f$ equals $6$
- From the graph $f(t) = 6$ at $t = -2$ and $t = 1;$ count solutions of $f(x) = -2$ and $f(x) = 1$
