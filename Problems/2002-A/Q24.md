# AMC12 2002-A Q24

![[assets/problems/2002-A/Q24.png]]

## Problem

Find the number of ordered pairs of real numbers $(a, b)$ such that $(a + bi)^{2002} = a - bi.$

(A) $1001$
(B) $1002$
(C) $2001$
(D) $2002$
(E) $2004$

## Answer

**E.** $2004$

## Solution

Let $z = a + bi.$ The equation is $z^{2002} = \overline{z}.$ Taking magnitudes, $|z|^{2002} = |z|,$ so $|z|\big(|z|^{2001} - 1\big) = 0,$ giving $|z| = 0$ or $|z| = 1.$

If $|z| = 0,$ then $(a, b) = (0, 0),$ one solution. If $|z| = 1,$ then $\overline{z} = \dfrac1z,$ so $z^{2002} = \dfrac1z,$ i.e. $z^{2003} = 1,$ which has $2003$ distinct roots.

Altogether there are $1 + 2003 = 2004$ ordered pairs.

Thus, the correct answer is **E**.

## Key idea

- Write $z = a + bi,$ so the equation is $z^{2002} = \overline{z};$ take magnitudes
- $|z|^{2002} = |z|$ forces $|z| = 0$ or $|z| = 1;$ when $|z| = 1,$ $\overline{z} = \dfrac1z$
