# AMC12 2011-A Q08

![[assets/problems/2011-A/Q08.png]]

## Problem

In the eight-term sequence $A, B, C, D, E, F, G, H,$ the value of $C$ is $5$ and the sum of any three consecutive terms is $30.$ What is $A + H?$

(A) $17$
(B) $18$
(C) $25$
(D) $26$
(E) $43$

## Answer

**C.** $25$

## Solution

Since $A + B + C = B + C + D = 30,$ we get $D = A,$ and likewise the sequence repeats with period $3.$ Thus $H,$ the eighth term, equals $B.$

From $A + B + C = 30$ and $C = 5,$ we have $A + H = A + B = 30 - 5 = 25.$

Thus, the correct answer is **C**.

## Knowledge points

- Pattern recognition
- Algebraic manipulation

## Key idea

Use the repeated relation among the eight terms to express the target sum in terms of the known values.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
