# AMC12 2011-A Q20

![[assets/problems/2011-A/Q20.png]]

## Problem

Let $f(x) = ax^2 + bx + c,$ where $a,$ $b,$ and $c$ are integers. Suppose that $f(1) = 0,$ $50 \lt f(7) \lt 60,$ $70 \lt f(8) \lt 80,$ and $5000k \lt f(100) \lt 5000(k+1)$ for some integer $k.$ What is $k?$

(A) $1$
(B) $2$
(C) $3$
(D) $4$
(E) $5$

## Answer

**C.** $3$

## Solution

Since $f(1) = a + b + c = 0,$ we have $c = -a - b.$ Then $$ f(7) = 48a + 6b = 6(8a + b), $$ $$ f(8) = 63a + 7b = 7(9a + b). $$

From $50 \lt 6(8a + b) \lt 60$ we get $8a + b = 9,$ and from $70 \lt 7(9a + b) \lt 80$ we get $9a + b = 11.$ Subtracting, $a = 2,$ then $b = -7$ and $c = 5.$

So $f(100) = 20000 - 700 + 5$ $= 19305,$ which lies in $5000 \cdot 3 \lt 19305 \lt 5000 \cdot 4,$ giving $k = 3.$

Thus, the correct answer is **C**.

## Knowledge points

- Quadratic
- System of equations
- Bounding to limit cases

## Key idea

With $f(1) = 0,$ the values $f(7) = 6(8a+b)$ and $f(8) = 7(9a+b)$ are pinned to single integers by their ranges, which determines $a,$ $b,$ $c.$

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
