# AMC12 2001 Q22

![[assets/problems/2001/Q22.png]]

## 题目

在矩形 $ABCD$ 中，点 $F$、$G$ 位于 $\overline{AB}$ 上，且 $AF = FG = GB$；点 $E$ 是 $\overline{DC}$ 的中点。此外，$\overline{AC}$ 分别与 $\overline{EF}$、$\overline{EG}$ 交于 $H$、$J$。矩形 $ABCD$ 的面积为 $70$。求三角形 $EHJ$ 的面积。

(A) $\dfrac{5}{2}$
(B) $\dfrac{35}{12}$
(C) $3$
(D) $\dfrac{7}{2}$
(E) $\dfrac{35}{8}$

## 答案

**C.** $3$

## 解答

三角形 $EFG$ 的底边 $FG = \dfrac{1}{3}AB$，高等于矩形的高，因此其面积为 $\dfrac{1}{6}(70) = \dfrac{35}{3}$。

由于 $EC \parallel AF$，三角形 $AFH$ 与 $CEH$ 相似，相似比为 $\dfrac{EC}{AF} = \dfrac{3}{2}$，故 $\dfrac{EH}{EF} = \dfrac{3}{5}$。同理，$\dfrac{EJ}{EG} = \dfrac{3}{7}$。

于是 $\dfrac{[EHJ]}{[EFG]}$ $= \dfrac{EH}{EF}\cdot\dfrac{EJ}{EG}$ $= \dfrac{3}{5}\cdot\dfrac{3}{7}$ $= \dfrac{9}{35}$，从而
$$
 [EHJ] = \dfrac{9}{35}\cdot\dfrac{35}{3} = 3.
$$

因此，正确答案是 **C**。

## 关键思路

- 三角形 $EFG$ 的底边为 $\dfrac{1}{3}AB$，高为矩形的全高，因此其面积为矩形面积的 $\dfrac{1}{6}$
- 利用 $EC \parallel AB$，由相似三角形得到 $\dfrac{EH}{EF} = \dfrac{3}{5}$ 和 $\dfrac{EJ}{EG} = \dfrac{3}{7}$

## 常见错误

- 计算前，先将题意转化为正确的方程、图形或计数模型。
- 检查单位、端点条件，以及所求量是数量、长度、面积还是概率。
