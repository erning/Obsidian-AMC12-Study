# AMC12 2001 Q17

![[assets/problems/2001/Q17.png]]

## 题目

从五边形内部随机选取一点 $P$。该五边形的顶点为 $A = (0, 2)$、$B = (4, 0)$、$C = (2\pi + 1, 0)$、$D = (2\pi + 1, 4)$、$E = (0, 4)$。$\angle APB$ 为钝角的概率是多少？

(A) $\dfrac{1}{5}$
(B) $\dfrac{1}{4}$
(C) $\dfrac{5}{16}$
(D) $\dfrac{3}{8}$
(E) $\dfrac{1}{2}$

## 答案

**C.** $\dfrac{5}{16}$

## 解答

当 $P$ 位于以 $AB$ 为直径的圆上时，$\angle APB = 90^\circ$。该圆的圆心为 $(2, 1)$，半径为 $\dfrac{|AB|}{2} = \dfrac{\sqrt{20}}{2} = \sqrt{5}$。当 $P$ 位于此圆内部时，该角为钝角。

相关的半圆完全位于五边形内，其面积为 $\dfrac{1}{2}\pi(\sqrt{5})^2 = \dfrac{5\pi}{2}$。

该五边形是以 $(0,0)$、$C$、$D$、$E$ 为顶点的矩形去掉三角形 $OAB$ 后所得，因此其面积为
$$
 4(2\pi + 1) - \dfrac{1}{2}(2)(4) = 8\pi.
$$

所求概率为 $\dfrac{5\pi/2}{8\pi} = \dfrac{5}{16}$。

因此，正确答案是 **C**。

## 关键思路

- 当且仅当 $P$ 位于以 $AB$ 为直径的圆内部时，$\angle APB$ 为钝角
- 该半圆完全位于五边形内，因此比较半圆面积与五边形面积即可

- 几何概率
- 圆周角

## 常见错误

- 计算前，先将题意转化为正确的方程、图形或计数模型。
- 检查单位、端点条件，以及所求量是数量、长度、面积还是概率。
