# AMC12 2001 Q19

![[assets/problems/2001/Q19.png]]

## 题目

多项式 $P(x) = x^3 + ax^2 + bx + c$ 满足：其零点的平均数、零点的乘积和各项系数之和都相等。若曲线 $y = P(x)$ 的 $y$ 轴截距为 $2$，则 $b$ 等于多少？

(A) $-11$
(B) $-10$
(C) $-9$
(D) $1$
(E) $5$

## 答案

**A.** $-11$

## 解答

$y$ 轴截距为 $P(0) = c = 2$。由韦达定理，零点之积为 $-c = -2$，零点的平均数为 $-\dfrac{a}{3}$，各项系数之和为 $P(1) = 1 + a + b + c$。

这三者均等于 $-2$。由 $-\dfrac{a}{3} = -2$，得 $a = 6$。

再由 $1 + a + b + c = -2$，可得 $1 + 6 + b + 2 = -2$，所以 $b = -11$。

因此，正确答案是 **A**。

## 关键思路

- 由韦达定理，零点之和为 $-a$，零点之积为 $-c$；各项系数之和为 $P(1)$
- $y$ 轴截距给出 $c = 2$，所以这三个量的公共值为零点之积

## 知识点

- 韦达定理
- 多项式

## 常见错误

- 计算前，先将题意转化为正确的方程、图形或计数模型。
- 检查单位、端点条件，以及所求量是数量、长度、面积还是概率。
