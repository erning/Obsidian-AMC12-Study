# AMC12 2001 Q13

![[assets/problems/2001/Q13.png]]

## Problem

The parabola with equation $y = ax^2 + bx + c$ and vertex $(h, k)$ is reflected about the line $y = k.$ This results in the parabola with equation $y = dx^2 + ex + f.$ Which of the following equals $a + b + c + d + e + f?$

(A) $2b$
(B) $2c$
(C) $2a + 2b$
(D) $2h$
(E) $2k$

## Answer

**E.** $2k$

## Solution

The value $a + b + c$ is the first parabola at $x = 1,$ and $d + e + f$ is the reflected parabola at $x = 1.$

Reflecting the curve about $y = k$ replaces each height $y$ by $2k - y.$ So at $x = 1$ the two heights sum to
$$
 (a + b + c) + (d + e + f) = 2k.
$$

Thus, the correct answer is **E**.

## Key idea

- Note that $a + b + c$ is the first parabola evaluated at $x = 1,$ and $d + e + f$ is the second at $x = 1$
- Reflecting about $y = k$ sends the value $y$ at each $x$ to $2k - y$
