# AMC12 2001 Q24

![[assets/problems/2001/Q24.png]]

## 题目

在三角形 $ABC$ 中，$\angle ABC = 45^\circ$。点 $D$ 位于 $\overline{BC}$ 上，且 $2 \cdot BD = CD$、$\angle DAB = 15^\circ$。求 $\angle ACB$。

(A) $54^\circ$
(B) $60^\circ$
(C) $72^\circ$
(D) $75^\circ$
(E) $90^\circ$

## 答案

**D.** $75^\circ$

## 解答

从 $C$ 向直线 $AD$ 作垂线，垂足为 $E$。由三角形 $ADB$ 的外角可得 $\angle ADC = 15^\circ + 45^\circ = 60^\circ$，所以三角形 $CDE$ 是 $30$-$60$-$90$ 三角形，且 $DE = \tfrac{1}{2}CD = BD$。

于是三角形 $BDE$ 为等腰三角形，$\angle EBD = \angle BED = 30^\circ$。又因 $\angle ECB = 30^\circ$，三角形 $BEC$ 为等腰三角形，故 $BE = EC$。

此外，$\angle ABE$ $= 45^\circ - 30^\circ$ $= 15^\circ$ $= \angle EAB$，故三角形 $ABE$ 为等腰三角形，$AE = BE$。因此 $AE = BE = EC$，直角三角形 $AEC$ 为等腰三角形，故 $\angle ECA = 45^\circ$。

所以 $\angle ACB$ $= \angle ECA + \angle ECD$ $= 45^\circ + 30^\circ$ $= 75^\circ$。

因此，正确答案是 **D**。

## 关键思路

- 由外角可得 $\angle ADC$ $= \angle DAB + \angle ABD$ $= 15^\circ + 45^\circ$ $= 60^\circ$
- 从 $C$ 向直线 $AD$ 作垂线，垂足为 $E$；此时三角形 $CDE$ 为 $30$-$60$-$90$ 三角形，且 $DE = \tfrac{1}{2}CD = BD$

- 计算前，先将题意转化为正确的方程、图形或计数模型。
- 检查单位、端点条件，以及所求量是数量、长度、面积还是概率。
