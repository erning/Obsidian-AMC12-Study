# AMC12 2001 Q06

![[assets/problems/2001/Q06.png]]

## 题目

一个电话号码的形式为 $ABC - DEF - GHIJ$，其中每个字母代表一个不同的数字。电话号码各部分中的数字均按递减顺序排列，即 $A \gt B \gt C$、$D \gt E \gt F$，且 $G \gt H \gt I \gt J$。此外，$D$、$E$、$F$ 是连续的偶数数字；$G$、$H$、$I$、$J$ 是连续的奇数数字；并且 $A + B + C = 9$。求 $A$。

(A) $4$
(B) $5$
(C) $6$
(D) $7$
(E) $8$

## 答案

**E.** $8$

## 解答

四个连续且递减的奇数数字 $GHIJ$ 只能是 $9753$ 或 $7531$，于是会剩下一个奇数数字（$1$ 或 $9$）给 $ABC$。

由于 $A + B + C = 9$，且 $ABC$ 的另外两个数字为偶数，所以该奇数数字必须是 $1$（若为 $9$，则两个偶数数字之和只能为 $0$）。故这两个偶数数字之和为 $8$。

三个连续且递减的偶数数字 $DEF$ 为 $864$、$642$ 或 $420$，于是留给 $ABC$ 的偶数对分别为 $\{2, 0\}$、$\{8, 0\}$ 或 $\{8, 6\}$。只有 $\{8, 0\}$ 的和为 $8$，所以 $ABC = 810$，即 $A = 8$。

因此，正确答案是 **E**。

## 关键思路

- 四个连续且递减的奇数数字只能为 $9753$ 或 $7531$
- 剩下的奇数数字位于 $ABC$ 中；结合另两个偶数数字之和为 $9$，该奇数数字只能为 $1$
