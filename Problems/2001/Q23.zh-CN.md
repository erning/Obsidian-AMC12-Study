# AMC12 2001 Q23

![[assets/problems/2001/Q23.png]]

## 题目

一个首项系数为 $1$、系数均为整数的四次多项式有两个实零点，且这两个零点均为整数。下列哪一项也可能是该多项式的零点？

(A) $\dfrac{1 + i\sqrt{11}}{2}$
(B) $\dfrac{1 + i}{2}$
(C) $\dfrac{1}{2} + i$
(D) $1 + \dfrac{i}{2}$
(E) $\dfrac{1 + i\sqrt{13}}{2}$

## 答案

**A.** $\dfrac{1 + i\sqrt{11}}{2}$

## 解答

设 $P(x)$ $= (x - r)(x - s)(x^2 + \alpha x + \beta)$，其中 $r$、$s$ 为整数根。比较系数可知，$\alpha$ 和 $\beta$ 均为整数。

另外两个零点为
$$
 -\dfrac{\alpha}{2} \pm \dfrac{i\sqrt{4\beta - \alpha^2}}{2}.
$$
实部为 $\dfrac{1}{2}$ 时，必须有 $\alpha = -1$，此时虚部为 $\dfrac{\sqrt{4\beta - 1}}{2}$。

选项 A 要求 $\sqrt{4\beta - 1} = \sqrt{11}$，即 $\beta = 3$，为整数，故可行。其余选项会使 $\beta$ 不是整数（例如选项 E 要求 $\beta = 3.5$；选项 D 要求 $\alpha = -2$ 且 $\beta = \tfrac{5}{4}$）。

因此，正确答案是 **A**。

## 关键思路

- 提取两个整数根后，剩余因子为 $x^2 + \alpha x + \beta$，其中 $\alpha$、$\beta$ 均为整数
- 复零点为 $-\dfrac{\alpha}{2} \pm \dfrac{i\sqrt{4\beta - \alpha^2}}{2}$；匹配实部后，检查 $\beta$ 是否为整数

## 知识点

- 多项式
- 复数
- 二次方程

## 常见错误

- 计算前，先将题意转化为正确的方程、图形或计数模型。
- 检查单位、端点条件，以及所求量是数量、长度、面积还是概率。
