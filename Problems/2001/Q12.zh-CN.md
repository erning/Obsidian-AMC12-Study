# AMC12 2001 Q12

![[assets/problems/2001/Q12.png]]

## 题目

不超过 $2001$ 的正整数中，是 $3$ 或 $4$ 的倍数但不是 $5$ 的倍数的有多少个？

(A) $768$
(B) $801$
(C) $934$
(D) $1067$
(E) $1167$

## 答案

**B.** $801$

## 解答

不超过 $2001$ 的 $3$ 或 $4$ 的倍数共有
$$
 667 + 500 - 166 = 1001,
$$
其中 $\lfloor 2001/3 \rfloor = 667$、$\lfloor 2001/4 \rfloor = 500$，且 $\lfloor 2001/12 \rfloor = 166$。

其中还能被 $5$ 整除的数是 $15$ 或 $20$ 的倍数，共有
$$
 133 + 100 - 33 = 200,
$$
其中 $\lfloor 2001/15 \rfloor = 133$、$\lfloor 2001/20 \rfloor = 100$，且 $\lfloor 2001/60 \rfloor = 33$。

所求个数为 $1001 - 200 = 801$。

因此，正确答案是 **B**。

## 关键思路

- 先用 $\lfloor 2001/3 \rfloor$ $+ \lfloor 2001/4 \rfloor$ $- \lfloor 2001/12 \rfloor$ 计算 $3$ 或 $4$ 的倍数个数
- 再减去同时为 $5$ 的倍数的数，即 $15$ 或 $20$ 的倍数
