# AMC12 2010-B Q13

![[assets/problems/2010-B/Q13.png]]

## Problem

In $\triangle ABC,$ $\cos(2A-B)+\sin(A+B)=2$ and $AB=4.$ What is $BC?$

(A) $\sqrt{2}$
(B) $\sqrt{3}$
(C) $2$
(D) $2\sqrt{2}$
(E) $2\sqrt{3}$

## Answer

**C.** $2$

## Solution

A cosine plus a sine equals $2$ only when each equals $1.$ So $\cos(2A-B)=1$ and $\sin(A+B)=1,$ giving $2A-B=0^\circ$ and $A+B=90^\circ.$

Solving, $A=30^\circ$ and $B=60^\circ,$ so $\triangle ABC$ is a $30\text{-}60\text{-}90$ right triangle with the right angle at $C.$

With hypotenuse $AB=4,$ the side $BC$ opposite the $30^\circ$ angle is half the hypotenuse, so $BC=2.$

Thus, the correct answer is **C**.

## Knowledge points

- Trigonometry
- Special right triangle
- Bounding to limit cases

## Key idea

A cosine plus a sine equal to $2$ forces both to equal $1,$ which fixes the angles; then read $BC$ off the $30\text{-}60\text{-}90$ triangle.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
