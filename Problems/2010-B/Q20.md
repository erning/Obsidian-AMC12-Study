# AMC12 2010-B Q20

![[assets/problems/2010-B/Q20.png]]

## Problem

A geometric sequence $(a_n)$ has $a_1=\sin x,$ $a_2=\cos x,$ and $a_3=\tan x$ for some real number $x.$ For what value of $n$ does $a_n=1+\cos x?$

(A) $4$
(B) $5$
(C) $6$
(D) $7$
(E) $8$

## Answer

**E.** $8$

## Solution

The common ratio is $r=\frac{a_2}{a_1}=\cot x.$ Then $a_4=a_3\cdot r=\tan x\cot x=1.$

From $a_3=a_1r^2,$ we get $\tan x=\sin x\cot^2 x=\frac{\cos^2 x}{\sin x},$ so $\sin^2 x=\cos^3 x,$ i.e. $(\cos^2 x)(1+\cos x)=1.$

Hence $1+\cos x=\frac{1}{\cos^2 x}.$ Also $r^2=\frac{\cos^2 x}{\sin^2 x}=\frac{\cos^2 x}{\cos^3 x}=\frac{1}{\cos x},$ so $r^4=\frac{1}{\cos^2 x}=1+\cos x.$

Therefore $1+\cos x=a_4\cdot r^4=a_8,$ so $n=8.$

Thus, the correct answer is **E**.

## Knowledge points

- Geometric sequence
- Trigonometric identity

## Key idea

Use the common ratio to derive $\sin^2 x = \cos^3 x,$ which rewrites $1 + \cos x$ as a power of the ratio times $a_4 = 1.$

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
