# AMC12 2016-B Q22

![[assets/problems/2016-B/Q22.png]]

## Problem

For $n<1000$, $1/n$ has repeating period $6$, while $1/(n+6)$ has period $4$. Locate $n$.

## Answer

**B.** $[201,400]$

## Solution

Period $6$ implies $n\mid10^6-1$. Period $4$ implies $n+6\mid10^4-1$, but it must not divide $10^2-1$. Hence $101\mid n+6$, so $n=101k-6$. Testing the possibilities below $1000$ leaves $n=297$, which lies in $[201,400]$.

## Knowledge points

- Repeating decimals
- Divisibility by $10^m-1$

## Key idea

Translate a repeating period into a divisibility condition.

## Common mistakes

- Allowing a smaller period that divides $4$
- Forgetting $n<1000$
