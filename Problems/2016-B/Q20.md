# AMC12 2016-B Q20

![[assets/problems/2016-B/Q20.png]]

## Problem

In a round-robin tournament, each team wins and loses $10$ games. How many triples form a cycle $A$ beats $B$, $B$ beats $C$, $C$ beats $A$?

## Answer

**A.** $385$

## Solution

There are $21$ teams, so ${21\choose3}=1330$ triples. A non-cyclic triple has one team beating both others. There are
$$21{10\choose2}=945$$
such triples. Therefore cyclic triples number
$$1330-945=385.$$

## Knowledge points

- Tournament counting
- Complementary casework

## Key idea

Count all triples and subtract transitive triples.

## Common mistakes

- Counting ordered triples instead of sets
- Assuming every triple is cyclic
