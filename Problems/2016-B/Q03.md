# AMC12 2016-B Q03

![[assets/problems/2016-B/Q03.png]]

## Problem

Let $x=-2016$. Evaluate $\left||x|-x\right|-|x|-x$.

## Answer

**D.** $4032$

## Solution

We have $|x|=2016$, so
$$|x|-x=4032.$$
Then
$$\left||x|-x\right|-|x|-x=4032-2016-(-2016)=4032.$$

## Knowledge points

- Absolute values
- Order of operations

## Key idea

Evaluate from the innermost absolute-value expression outward.

## Common mistakes

- Replacing $x$ by $|x|$ everywhere
- Losing the final minus sign before $x$
