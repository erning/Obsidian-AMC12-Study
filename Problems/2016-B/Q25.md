# AMC12 2016-B Q25

![[assets/problems/2016-B/Q25.png]]

## Problem

The sequence has $a_0=1$, $a_1=\sqrt[19]{2}$, and $a_n=a_{n-1}a_{n-2}^2$. Find the least $k$ for which $a_1a_2\cdots a_k$ is an integer.

## Answer

**A.** $17$

## Solution

Write $a_n=2^{b_n/19}$. Then $b_0=0$, $b_1=1$, and
$$b_n=b_{n-1}+2b_{n-2}.$$
The solution is
$$b_n=\frac{2^n-(-1)^n}{3}.$$
For odd $k$, the exponent of $2$ in the product is integral exactly when $19\mid2^{k+1}-1$. The order of $2$ modulo $19$ is $18$, so the least odd $k$ has $k+1=18$, giving $k=17$.

## Knowledge points

- Recursive sequences
- Modular order

## Key idea

Track the exponent of $2$ rather than the radicals themselves.

## Common mistakes

- Multiplying radicals directly
- Checking only $a_k$ instead of the whole product
