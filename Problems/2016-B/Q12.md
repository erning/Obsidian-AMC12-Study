# AMC12 2016-B Q12

![[assets/problems/2016-B/Q12.png]]

## Problem

Numbers $1$ through $9$ fill a $3\times3$ grid so consecutive numbers share an edge. The four corners sum to $18$. Find the center.

## Answer

**C.** $7$

## Solution

Color the grid like a checkerboard. The four corners and center have the same color. Since consecutive numbers lie on opposite colors, that color contains the five odd numbers, whose sum is $25$. Therefore the center is
$$25-18=7.$$

## Knowledge points

- Checkerboard coloring
- Parity

## Key idea

Use the parity forced by adjacency.

## Common mistakes

- Assuming the center is $5$ by symmetry
- Using ordinary geometric adjacency instead of edge adjacency
