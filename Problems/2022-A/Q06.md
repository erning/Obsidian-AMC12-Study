# AMC12 2022-A Q06

![[assets/problems/2022-A/Q06.png]]

## Problem

A data set consists of six positive integers, not necessarily distinct: $1,7,5,2,5,$ and $X$. The average of the six numbers equals a value in the data set. What is the sum of all positive values of $X$?

## Answer

- Correct choice: **(D) $36$**

## Solution

The five known numbers sum to
$$
1+7+5+2+5=20,
$$
so the average is
$$
\frac{20+X}{6}.
$$
This must equal one of the values in the data set:

- If it equals $5$, then $20+X=30$, so $X=10$.
- If it equals $7$, then $20+X=42$, so $X=22$.
- If it equals $X$, then $20+X=6X$, so $X=4$.

Setting the average equal to $1$ or $2$ gives negative $X$, which is not allowed. Hence the sum of all positive values is
$$
10+22+4=\boxed{36}.
$$

## Key idea

The average must equal one of the listed data values, including possibly $X$ itself.

## Common mistakes

- Forgetting to test the case in which the average equals $X$.
