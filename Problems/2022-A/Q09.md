# AMC12 2022-A Q09

![[assets/problems/2022-A/Q09.png]]

## Problem

On Halloween, $31$ children entered the principal's office. Some always lie, some always tell the truth, and some alternate between lying and telling the truth. The principal asked everyone:

1. “Are you a truth-teller?” The answer was yes from $22$ children.
2. “Are you an alternater?” The answer was yes from $15$ children.
3. “Are you a liar?” The answer was yes from $9$ children.

How many pieces of candy did the principal give to the children who always tell the truth?

## Answer

- Correct choice: **(A) $7$**

## Solution

An alternater who begins with a lie answers yes to all three questions: the three propositions have truth values false, true, false, and the responses have truth values lie, truth, lie. An alternater who begins with the truth answers no to all three.

The third question receives yes from exactly the lie-first alternaters, so there are $9$ of them. The second question receives yes from all liars and those $9$ alternaters, so the number of liars is
$$
15-9=6.
$$
The first question receives yes from all truth-tellers, all liars, and the $9$ lie-first alternaters. Hence the number of truth-tellers is
$$
22-6-9=7.
$$
Each truth-teller receives candy only on the first question, so the total is $\boxed{7}$.

## Knowledge points

- Logic
- Truth tables
- Counting by types

## Key idea

Use the third answer count to identify one type of alternater first.

## Common mistakes

- Forgetting that an alternater's response pattern depends on the truth value of the proposition.
