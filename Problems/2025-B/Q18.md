# AMC12 2025-B Q18

![[assets/problems/2025-B/Q18.png]]

## Problem

Awnik repeatedly plays a game with probability $\frac13$ of winning. The outcomes are independent. What is the expected number of games he will play until he has both won and lost at least once?

## Answer

- Correct choice: **(D) $\frac72$**

## Solution

The first game is always played. After it, there are two cases:

- With probability $\frac13$, the first game is a win. The expected number of additional games until the first loss is $\frac{1}{2/3}=\frac32$.
- With probability $\frac23$, the first game is a loss. The expected number of additional games until the first win is $\frac{1}{1/3}=3$.

Therefore

$$
\begin{aligned}
E
&=1+\frac13\cdot\frac32+\frac23\cdot3\\
&=\boxed{\frac72}.
\end{aligned}
$$

## Knowledge points

- Expected value
- Geometric distribution
- Conditioning

## Key idea

Condition on the first game and wait only for the missing outcome.

## Common mistakes

- Forgetting the first game.
- Using $3$ as the waiting time for a loss after a win.
- Treating the two cases as equally likely.
