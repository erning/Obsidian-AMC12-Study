# AMC12 2025-B Q19

![[assets/problems/2025-B/Q19.png]]

## Problem

A rectangular grid has $141$ rows and $91$ columns. Horace fills the grid horizontally with the numbers from $1$ through $141\cdot91$, while Vera fills it vertically with the same numbers. How many squares receive two copies of the same number?

## Answer

- Correct choice: **(C) $11$**

## Solution

Index rows by $r$ and columns by $c$, where $1\le r\le141$ and $1\le c\le91$.

Horace writes

$$
H(r,c)=91(r-1)+c,
$$

while Vera writes

$$
V(r,c)=141(c-1)+r.
$$

Set these equal:

$$
91(r-1)+c=141(c-1)+r.
$$

After simplifying,

$$
90r-140c=-50,
\qquad
9r=14c-5.
$$

Modulo $9$,

$$
5c-5\equiv0\pmod9,
$$

so $c\equiv1\pmod9$. The possible columns are

$$
1,10,19,\ldots,91,
$$

which gives $11$ values. Each produces a valid row $r=(14c-5)/9$, so the answer is $\boxed{11}$.

## Knowledge points

- Coordinate indexing
- Diophantine equations
- Modular arithmetic

## Key idea

Express both fillings as linear functions of the row and column, then solve the equality in integers.

## Common mistakes

- Swapping the row and column counts.
- Mixing zero-based and one-based indexing.
- Counting possible columns without checking the corresponding rows.
