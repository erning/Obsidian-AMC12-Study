# AMC12 2025-B Q24

![[assets/problems/2025-B/Q24.png]]

## Problem

How many real numbers satisfy

$$
\sin(20\pi x)=\log_{20}x?
$$

## Answer

- Correct choice: **(D) $399$**

## Solution

Because the sine function is between $-1$ and $1$, any solution must satisfy

$$
-1\le\log_{20}x\le1,
$$

which is equivalent to

$$
\frac1{20}\le x\le20.
$$

On this interval, $\log_{20}x$ is strictly increasing from $-1$ to $1$. The sine function has period

$$
\frac{2\pi}{20\pi}=\frac1{10},
$$

and each monotone half-wave has length $\frac1{20}$. On each complete half-wave, the oscillating sine curve crosses the slowly increasing logarithm exactly once. Counting the half-waves in the interval gives $398$ complete branches, together with one additional crossing on the partial branch at the left endpoint. The final partial branch contributes no extra crossing.

Hence the total number of solutions is

$$
398+1=\boxed{399}.
$$

## Knowledge points

- Range restrictions
- Logarithmic functions
- Counting intersections of periodic and monotone functions

## Key idea

First restrict the domain using the range of sine, then count one intersection on each relevant monotone half-wave.

## Common mistakes

- Counting over all real $x$ without applying the logarithm/sine range restriction.
- Using the period $\frac1{20}$ instead of the half-wave length $\frac1{20}$ after distinguishing full periods.
- Counting endpoint intersections twice.
