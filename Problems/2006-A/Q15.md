# AMC12 2006-A Q15

![[assets/problems/2006-A/Q15.png]]

## Problem

Suppose $\cos x = 0$ and $\cos(x + z) = \tfrac{1}{2}.$ What is the smallest possible positive value of $z?$

(A) $\dfrac{\pi}{6}$
(B) $\dfrac{\pi}{3}$
(C) $\dfrac{\pi}{2}$
(D) $\dfrac{5\pi}{6}$
(E) $\dfrac{7\pi}{6}$

## Answer

**A.** $\dfrac{\pi}{6}$

## Solution

Since $\cos x = 0,$ we have $x = \tfrac{\pi}{2} + k\pi.$ Since $\cos(x + z) = \tfrac{1}{2},$ we have $x + z = 2n\pi \pm \tfrac{\pi}{3}.$

Taking $x = -\tfrac{\pi}{2}$ and $x + z = -\tfrac{\pi}{3}$ gives $z = -\tfrac{\pi}{3} + \tfrac{\pi}{2} = \tfrac{\pi}{6},$ the smallest positive value.

Thus, the correct answer is **A**.
