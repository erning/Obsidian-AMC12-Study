# AMC12 2006-A Q11

![[assets/problems/2006-A/Q11.png]]

## Problem

Which of the following describes the graph of the equation $(x + y)^2 = x^2 + y^2?$

(A) the empty set
(B) one point
(C) two lines
(D) a circle
(E) the entire plane

## Answer

**C.** two lines

## Solution

Expanding, $x^2 + 2xy + y^2 = x^2 + y^2,$ so $2xy = 0,$ i.e. $xy = 0.$

This is the union of the two coordinate axes, a pair of lines.

Thus, the correct answer is **C**.
