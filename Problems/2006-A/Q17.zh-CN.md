# AMC12 2006-A Q17

![[assets/problems/2006-A/Q17.png]]

## 题目

正方形 $ABCD$ 的边长为 $s$，以 $E$ 为圆心的圆半径为 $r$，且 $r$ 与 $s$ 均为有理数。圆经过 $D$，且 $D$ 位于 $\overline{BE}$ 上。点 $F$ 位于圆上，并与 $A$ 在 $\overline{BE}$ 的同侧。线段 $AF$ 是圆的切线，且 $AF = \sqrt{9 + 5\sqrt{2}}$。$r/s$ 是多少？

(A) $\dfrac{1}{2}$
(B) $\dfrac{5}{9}$
(C) $\dfrac{3}{5}$
(D) $\dfrac{5}{3}$
(E) $\dfrac{9}{5}$

## 答案

**B.** $\dfrac{5}{9}$

## 解答

取 $B = (0, 0),\ C = (s, 0),\ A = (0, s),\ D = (s, s)$，则位于射线 $BD$ 上的 $E = \left(s + \tfrac{r}{\sqrt{2}},\ s + \tfrac{r}{\sqrt{2}}\right)$。

由于 $AF$ 为圆的切线，$AF^2 = AE^2 - r^2$。计算 $AE^2$ 并化简，得 $9 + 5\sqrt{2} = s^2 + rs\sqrt{2}$。

由于 $r$ 和 $s$ 都是有理数，有理部分和无理部分分别相等：$s^2 = 9$ 且 $rs = 5$。因此 $s = 3,\ r = \tfrac{5}{3}$，从而 $r/s = \tfrac{5}{9}$。

因此，正确答案为 **B**。

## 知识点

- 点的幂
- 切线
- 坐标几何

## 关键思路

选取坐标，使直线与圆的方程尽量简单。

## 常见错误

- 在确定题目给出的量前就套用公式。
- 计算过程中未持续注意单位、符号和端点条件。
