# AMC12 2006-A Q18

![[assets/problems/2006-A/Q18.png]]

## 题目

函数 $f$ 具有如下性质：对其定义域内每个实数 $x$，$1/x$ 也在其定义域内，且
$$
 f(x) + f\!\left(\frac{1}{x}\right) = x.
$$
函数 $f$ 的定义域最多可以包含哪些实数？

(A) $\{x \mid x \neq 0\}$
(B) $\{x \mid x \lt 0\}$
(C) $\{x \mid x \gt 0\}$
(D) $\{x \mid x \neq -1,\ x \neq 0,\ x \neq 1\}$
(E) $\{-1, 1\}$

## 答案

**E.** $\{-1, 1\}$

## 解答

以 $1/x$ 替换 $x$，得 $f\!\left(\tfrac{1}{x}\right) + f(x) = \tfrac{1}{x}$。与 $f(x) + f\!\left(\tfrac{1}{x}\right) = x$ 联立，要求 $x = \tfrac{1}{x}$，所以 $x = \pm 1$。

两个值均满足条件，取 $f(1) = \tfrac{1}{2}$ 和 $f(-1) = -\tfrac{1}{2}$ 即可。因此最大的可能定义域为 $\{-1, 1\}$。

因此，正确答案为 **E**。
