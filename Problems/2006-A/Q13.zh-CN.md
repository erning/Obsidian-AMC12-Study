# AMC12 2006-A Q13

![[assets/problems/2006-A/Q13.png]]

## 题目

如图，一个 $3$–$4$–$5$ 直角三角形的顶点分别是三个两两外切圆的圆心。这些圆的面积之和是多少？

(A) $12\pi$
(B) $\dfrac{25\pi}{2}$
(C) $13\pi$
(D) $\dfrac{27\pi}{2}$
(E) $14\pi$

## 答案

**E.** $14\pi$

## 解答

设三个顶点处圆的半径为 $r, s, t$，则 $r + s = 3,\ r + t = 4,\ s + t = 5$。三式相加得 $r + s + t = 6$，所以 $r = 1,\ s = 2,\ t = 3$。

面积之和为 $\pi(1^2 + 2^2 + 3^2) = 14\pi$。

因此，正确答案为 **E**。

## 知识点

- 相切圆
- 方程组
- 圆的面积

## 关键思路

利用公切关系及圆心距建立半径关系。

## 常见错误

- 在确定题目给出的量前就套用公式。
- 计算过程中未持续注意单位、符号和端点条件。
