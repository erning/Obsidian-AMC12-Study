# AMC12 2006-A Q10

![[assets/problems/2006-A/Q10.png]]

## Problem

For how many real values of $x$ is $\sqrt{120 - \sqrt{x}}$ an integer?

(A) $3$
(B) $6$
(C) $9$
(D) $10$
(E) $11$

## Answer

**E.** $11$

## Solution

Let $k = \sqrt{120 - \sqrt{x}}$ be an integer. Then $k \ge 0$ and $k^2 = 120 - \sqrt{x} \le 120,$ so $0 \le k \le 10.$

Each such $k$ gives $\sqrt{x} = 120 - k^2 \ge 0,$ hence a distinct value $x = (120 - k^2)^2.$ That is $11$ values.

Thus, the correct answer is **E**.

- Radical
- Counting integers in a range

## Key idea

Compare the radical expression by squaring or rationalizing carefully.
