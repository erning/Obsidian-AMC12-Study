# AMC12 2008-A Q17

![[assets/problems/2008-A/Q17.png]]

## 题目

设整数数列 $a_1, a_2, \ldots$ 由如下规则确定：若 $a_{n-1}$ 为偶数，则 $a_n = a_{n-1}/2$；若 $a_{n-1}$ 为奇数，则 $a_n = 3a_{n-1} + 1$。在满足 $a_1 \le 2008$ 的正整数 $a_1$ 中，有多少个使得 $a_1$ 小于 $a_2, a_3$ 和 $a_4$ 中的每一个？

(A) $250$
(B) $251$
(C) $501$
(D) $502$
(E) $1004$

## 答案

**D.** $502$

## 解答

若 $a_1$ 为偶数，则 $a_2 = a_1/2 \lt a_1$，不满足条件。

若 $a_1 \equiv 1 \pmod 4$，则 $a_2 = 3a_1 + 1$ 是 $4$ 的倍数，所以 $a_3 = (3a_1 + 1)/2$、$a_4 = (3a_1 + 1)/4 \le a_1$，仍不满足条件。

若 $a_1 \equiv 3 \pmod 4$，则 $a_2$ 是偶数但不是 $4$ 的倍数，所以 $a_3 = (3a_1 + 1)/2 \gt a_1$；且 $a_3$ 为奇数，于是 $a_4 = 3a_3 + 1 \gt a_3 \gt a_1$。条件成立。

满足 $a_1 \equiv 3 \pmod 4$ 且 $a_1 \le 2008$ 的 $a_1$ 恰有 $\tfrac{2008}{4} = 502$ 个。

所以，正确答案为 **D**。

## 知识点

- 递推
- 模运算
- 分类讨论

## 核心思路

根据递推关系按相关模数分类，逐一枚举可能情形。

## 常见错误

- 在尚未确定相关量和条件前就套用公式。
- 计入无效情形，或忽略题目给出的边界条件。
