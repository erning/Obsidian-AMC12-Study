# AMC12 2008-A Q12

![[assets/problems/2008-A/Q12.png]]

## Problem

A function $f$ has domain $[0, 2]$ and range $[0, 1].$ (The notation $[a, b]$ denotes $\{x : a \le x \le b\}.$) What are the domain and range, respectively, of the function $g$ defined by $$g(x) = 1 - f(x + 1)?$$

(A) [-1,1],[-1,0]
(B) [-1,1],[0,1]
(C) [0,2],[-1,0]
(D) [1,3],[-1,0]
(E) [1,3],[0,1]

## Answer

**B.** [-1,1],[0,1]

## Solution

The value $f(x + 1)$ is defined when $0 \le x + 1 \le 2,$ that is, $-1 \le x \le 1,$ so the domain of $g$ is $[-1, 1].$

As $f(x + 1)$ ranges over $[0, 1],$ the value $1 - f(x + 1)$ ranges over $[0, 1]$ as well, so the range of $g$ is $[0, 1].$

Thus, **B** is the correct answer.

## Key idea

Use the definition of the function and its domain/range constraints to determine which listed interval pair is possible.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
