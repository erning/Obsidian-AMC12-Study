# AMC12 2008-A Q19

![[assets/problems/2008-A/Q19.png]]

## 题目

在 $$ \begin{aligned} &\left(1 + x + x^2 + \cdots + x^{27}\right) \\ &\quad {}\cdot \left(1 + x + x^2 + \cdots + x^{14}\right)^2, \end{aligned} $$ 的展开式中，$x^{28}$ 的系数是多少？

(A) $195$
(B) $196$
(C) $224$
(D) $378$
(E) $405$

## 答案

**C.** $224$

## 解答

每一项均为 $x^{a + b + c}$，其中 $0 \le a \le 27$，且 $0 \le b, c \le 14$。要得到 $x^{28}$，需有 $a = 28 - b - c$。

$(b, c)$ 有 $(14 + 1)^2 = 225$ 种取法。除 $(b, c) = (0, 0)$ 外，每种取法所需的 $a = 28 - b - c$ 都在 $[0, 27]$ 内，因而对应一个有效项。

故 $x^{28}$ 的系数为 $225 - 1 = 224$。

所以，正确答案为 **C**。

## 知识点

- 生成函数
- 分拆与组合

## 核心思路

将乘积视为生成函数，提取目标幂次的系数。

## 常见错误

- 在尚未确定相关量和条件前就套用公式。
- 计入无效情形，或忽略题目给出的边界条件。
