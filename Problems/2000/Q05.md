## Problem

If $|x - 2| = p,$ where $x \lt 2,$ then what is $x - p?$

(A) $-2$
(B) $2$
(C) $2 - 2p$
(D) $2p - 2$
(E) $|2p - 2|$

## Answer

**C.** $2 - 2p$

## Solution

Since $x \lt 2,$ we have $|x - 2| = 2 - x = p,$ so $x = 2 - p.$

Then
$$
 x - p = (2 - p) - p = 2 - 2p.
$$

Thus, the correct answer is **C**.

## Key idea

- Since $x \lt 2,$ the quantity $x - 2$ is negative, so $|x - 2| = 2 - x$
- Solve $2 - x = p$ for $x,$ then compute $x - p$
