# AMC12 2000 Q07

![[assets/problems/2000/Q07.png]]

## Problem

How many positive integers $b$ have the property that $\log_b 729$ is a positive integer?

(A) $0$
(B) $1$
(C) $2$
(D) $3$
(E) $4$

## Answer

**E.** $4$

## Solution

If $\log_b 729 = n,$ then $b^n = 729 = 3^6,$ so $n$ must be a positive divisor of $6.$

The possibilities $n = 1, 2, 3, 6$ give $b = 3^6, 3^3, 3^2, 3^1,$ that is, $729, 27, 9,$ and $3.$

There are $4$ such values of $b.$

Thus, the correct answer is **E**.

## Key idea

- If $\log_b 729 = n,$ then $b^n = 729 = 3^6$
- For $b$ to be an integer, $n$ must be a positive divisor of $6$
