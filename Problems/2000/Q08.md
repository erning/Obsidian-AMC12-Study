# AMC12 2000 Q08

![[assets/problems/2000/Q08.png]]

## Problem

Figures $0, 1, 2,$ and $3$ consist of $1, 5, 13,$ and $25$ nonoverlapping unit squares, respectively. If the pattern were continued, how many nonoverlapping unit squares would there be in figure $100?$

(A) $10401$
(B) $19801$
(C) $20201$
(D) $39801$
(E) $40801$

## Answer

**C.** $20201$

## Solution

Figure $n$ is a diamond whose row lengths increase through the odd numbers and back down, giving a total of $n^2 + (n + 1)^2$ unit squares. This matches $1, 5, 13, 25$ for $n = 0, 1, 2, 3.$

Therefore figure $100$ has
$$
 \begin{aligned} 100^2 + 101^2 &= 10000 + 10201 \\ &= 20201 \end{aligned}
$$
 unit squares.

Thus, the correct answer is **C**.

## Key idea

- Each figure is a diamond whose rows are the odd numbers $1, 3, 5, \ldots$ up and back down
- Figure $n$ has $n^2 + (n + 1)^2$ unit squares

- Pattern recognition
- Perfect square

## Common mistakes

- Translate the wording into the correct equation, diagram, or counting model before calculating.
- Check units, endpoint conditions, and whether the requested quantity is a count, length, area, or probability.
