# AMC12 2000 Q15

![[assets/problems/2000/Q15.png]]

## Problem

Let $f$ be a function for which $f\!\left(\dfrac{x}{3}\right) = x^2 + x + 1.$ Find the sum of all values of $z$ for which $f(3z) = 7.$

(A) $-\dfrac{1}{3}$
(B) $-\dfrac{1}{9}$
(C) $0$
(D) $\dfrac{5}{9}$
(E) $\dfrac{5}{3}$

## Answer

**B.** $-\dfrac{1}{9}$

## Solution

Setting $\dfrac{x}{3} = 3z$ gives $x = 9z,$ so
$$
 \begin{aligned} f(3z) &= (9z)^2 + 9z + 1 \\ &= 81z^2 + 9z + 1 \\ &= 7. \end{aligned}
$$

This rearranges to $81z^2 + 9z - 6 = 0.$

By the sum-of-roots formula, the sum of the values of $z$ is $-\dfrac{9}{81} = -\dfrac{1}{9}.$

Thus, the correct answer is **B**.

## Key idea

- To evaluate $f(3z),$ set $\dfrac{x}{3} = 3z,$ so $x = 9z$
- This gives a quadratic in $z$; use the sum-of-roots formula $-\dfrac{b}{a}$
