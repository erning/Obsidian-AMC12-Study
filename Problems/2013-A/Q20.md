# AMC12 2013-A Q20

![[assets/problems/2013-A/Q20.png]]

## Problem

Let $S=\{1,2,3,\ldots,19\}$. For $a,b\in S$, define $a\succ b$ to mean that either $0<a-b\le9$ or $b-a>9$. How many ordered triples $(x,y,z)$ satisfy $x\succ y$, $y\succ z$, and $z\succ x$?

## Answer

**B.** $855$

## Solution

Read the elements modulo $19$. The relation $a\succ b$ is equivalent to
$$0<(a-b)\bmod19\le9.$$

For a cyclic triple, define
$$u=(x-y)\bmod19,\quad v=(y-z)\bmod19,\quad w=(z-x)\bmod19.$$
Each of $u,v,w$ is in ${1,2,\ldots,9}$, and
$$u+v+w\equiv0\pmod{19}.$$
Because $3\le u+v+w\le27$, the sum must equal $19$. The number of ordered positive triples at most $9$ with sum $19$ is $45$.

For each such difference triple, there are $19$ choices for $x$, after which $y$ and $z$ are determined. Therefore the total is
$$19\cdot45=855.$$

## Knowledge points

- Modular arithmetic
- Cyclic relations
- Bounded compositions

## Key idea

Translate the relation into positive modular differences whose sum must be $19$.

## Common mistakes

- Treating $a\succ b$ as an ordinary linear order
- Counting unordered triples instead of ordered triples
