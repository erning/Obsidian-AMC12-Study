# AMC12 2013-A Q13

![[assets/problems/2013-A/Q13.png]]

## Problem

Let $A=(0,0)$, $B=(1,2)$, $C=(3,3)$, and $D=(4,0)$. Quadrilateral $ABCD$ is cut into equal-area pieces by a line passing through $A$. This line intersects $\overline{CD}$ at $\left(\frac pq,\frac rs\right)$, where the fractions are in lowest terms. What is $p+q+r+s$?

## Answer

**B.** $58$

## Solution

The shoelace formula gives
$$[ABCD]=\frac{15}{2},$$
so each piece has area $\frac{15}{4}$. Let the line meet $CD$ at $G$. The piece $ADG$ has base $AD=4$, so
$$\frac12\cdot4\cdot y_G=\frac{15}{4},$$
which gives $y_G=\frac{15}{8}$.

Line $CD$ has equation $y=-3(x-4)$. Substituting $y_G$ gives
$$x_G=4-\frac{1}{3}\cdot\frac{15}{8}=\frac{27}{8}.$$
Thus
$$p+q+r+s=27+8+15+8=58.$$

## Knowledge points

- Shoelace formula
- Coordinate geometry
- Triangle area

## Key idea

Use the equal-area condition to determine the height of the intersection point before using the line equation.

## Common mistakes

- Computing the quadrilateral area incorrectly
- Using the full quadrilateral as the equal-area subregion
