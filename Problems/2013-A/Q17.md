# AMC12 2013-A Q17

![[assets/problems/2013-A/Q17.png]]

## Problem

A group of $12$ pirates divide a treasure as follows: the $k$th pirate takes $\frac{k}{12}$ of the coins remaining in the chest. The initial number of coins is the smallest number for which every pirate receives a positive whole number of coins. How many coins does the $12$th pirate receive?

## Answer

**D.** $1925$

## Solution

Let $n$ be the number of coins received by the $12$th pirate. Reversing the process, before pirate $k$ takes a share, the number of coins is multiplied by
$$\frac{12}{12-k}.$$
After reversing the first $11$ steps, the initial number is
$$\frac{12^{11}n}{11!}.
$$
Since
$$\frac{12^{11}}{11!}=\frac{2^{14}\cdot3^7}{5^2\cdot7\cdot11},$$
the smallest positive $n$ making the initial count an integer is
$$n=5^2\cdot7\cdot11=1925.$$
With this value, the earlier shares are also integers, so the $12$th pirate receives $1925$ coins.

## Knowledge points

- Divisibility
- Factorials
- Reverse recurrence

## Key idea

Reverse the successive fractional withdrawals and choose the smallest final remainder that clears all prime denominators.

## Common mistakes

- Multiplying by $k/12$ while reversing the process
- Checking only that the initial total is an integer
