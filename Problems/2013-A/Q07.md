# AMC12 2013-A Q07

![[assets/problems/2013-A/Q07.png]]

## Problem

The sequence $S_1,S_2,S_3,\ldots,S_{10}$ has the property that every term beginning with the third is the sum of the previous two:
$$S_n=S_{n-2}+S_{n-1}\quad(n\ge3).$$
Suppose that $S_9=110$ and $S_7=42$. What is $S_4$?

## Answer

**C.** $10$

## Solution

Work backward through the recurrence:
$$S_8=S_9-S_7=110-42=68,$$
$$S_6=S_8-S_7=68-42=26,$$
$$S_5=S_7-S_6=42-26=16,$$
and
$$S_4=S_6-S_5=26-16=10.$$

## Knowledge points

- Linear recurrences
- Backward substitution

## Key idea

The recurrence can be reversed using $S_{n-2}=S_n-S_{n-1}$.

## Common mistakes

- Computing forward from insufficient information
- Subtracting the terms in the wrong order
