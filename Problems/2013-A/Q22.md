# AMC12 2013-A Q22

![[assets/problems/2013-A/Q22.png]]

## Problem

A palindrome is a nonnegative integer that reads the same forwards and backwards in base $10$, with no leading zeros. A $6$-digit palindrome $n$ is chosen uniformly at random. What is the probability that $\frac n{11}$ is also a palindrome?

## Answer

**E.** $\frac{11}{30}$

## Solution

Every six-digit palindrome is divisible by $11$. Let
$$m=\frac n{11}.$$
A four-digit palindrome for $m$ cannot produce a six-digit palindrome after multiplication by $11$, so write
$$m=\overline{abcba},$$
where $a\ne0$.

Multiplying by $11$ produces a six-digit palindrome exactly when no carries occur across the symmetric digit pairs. This requires
$$a+b\le9,qquad b+c\le9.$$
For a fixed $b$, there are $9-b$ choices for $a$ and $10-b$ choices for $c$. Thus the number of favorable palindromes is
$$
\sum_{b=0}^{9}(9-b)(10-b)=330.
$$
There are $9\cdot10^2=900$ six-digit palindromes in total. Hence
$$\frac{330}{900}=\frac{11}{30}.$$

## Knowledge points

- Palindromes
- Multiplication with carries
- Counting

## Key idea

Write the quotient as a five-digit palindrome and characterize exactly when multiplication by $11$ creates no carries.

## Common mistakes

- Counting all six-digit palindromes as $10^3$
- Ignoring the leading-digit restriction
