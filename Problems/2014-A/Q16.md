# AMC12 2014-A Q16

![[assets/problems/2014-A/Q16.png]]

## Problem

The product $8(888\ldots8)$, where the second factor has $k$ digits, is an integer whose digits have sum $1000$. What is $k$?

## Answer

**D.** $991$

## Solution

Multiplying by $8$ gives a string with the pattern
$$8(888\ldots8)=7\underbrace{11\ldots1}_{k-2\text{ digits}}04.$$
Its digit sum is
$$7+(k-2)+0+4=k+9.$$
Therefore
$$k+9=1000,\qquad k=991.$$

## Knowledge points

- Multiplication with carries
- Digit sums

## Key idea

Recognize the repeating digit pattern produced by multiplying a string of eights by $8$.

## Common mistakes

- Assuming the product is simply a string of sixes
- Miscounting the number of middle ones
