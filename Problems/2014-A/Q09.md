# AMC12 2014-A Q09

![[assets/problems/2014-A/Q09.png]]

## Problem

Five positive consecutive integers starting with $a$ have average $b$. What is the average of five consecutive integers starting with $b$?

## Answer

**B.** $a+4$

## Solution

The first five integers are
$$a,a+1,a+2,a+3,a+4,$$
so their average is
$$b=a+2.$$
The next five integers start at $b$, so their average is $b+2=a+4$.

## Knowledge points

- Arithmetic sequences
- Averages

## Key idea

The average of consecutive integers is the middle integer.

## Common mistakes

- Adding $5$ instead of $2$ to the starting value
