# AMC12 2014-A Q18

![[assets/problems/2014-A/Q18.png]]

## Problem

The domain of
$$f(x)=\log_{1/2}\left(\log_4\left(\log_{1/4}\left(\log_{16}\left(\log_{1/16}x\right)\right)\right)\right)$$
is an interval of length $\frac mn$, where $m,n$ are relatively prime positive integers. What is $m+n$?

## Answer

**C.** $271$

## Solution

Working from the outside, the argument of the outer logarithm must be positive:
$$\log_4\left(\log_{1/4}\left(\log_{16}\left(\log_{1/16}x\right)\right)\right)>0.$$
Thus
$$\log_{1/4}\left(\log_{16}\left(\log_{1/16}x\right)\right)>1.$$
Because the base $1/4$ is less than $1$, this is equivalent to
$$0<\log_{16}\left(\log_{1/16}x\right)<\frac14.$$
Therefore
$$1<\log_{1/16}x<2.$$
Since the base $1/16$ is less than $1$, the domain is
$$\frac1{256}<x<\frac1{16},$$
whose length is
$$\frac1{16}-\frac1{256}=\frac{15}{256}.$$
Thus $m+n=15+256=271.$$

## Knowledge points

- Domains of nested logarithms
- Reversing inequalities for bases below $1$

## Key idea

Track the positivity constraint from the outside inward, reversing inequalities whenever the logarithm base is below $1$.

## Common mistakes

- Forgetting that every nested logarithm needs a positive argument
- Not reversing an inequality for base $1/4$ or $1/16$
