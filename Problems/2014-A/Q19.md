# AMC12 2014-A Q19

![[assets/problems/2014-A/Q19.png]]

## Problem

There are exactly $N$ distinct rational numbers $k$ such that $|k|<200$ and
$$5x^2+kx+12=0$$
has at least one integer solution for $x$. What is $N$?

## Answer

**E.** $78$

## Solution

If $x$ is an integer root, then $x\ne0$ and
$$k=-5x-\frac{12}{x}.$$
For $x>0$, $|k|=5x+12/x$. This is less than $200$ for $1\le x\le39$, while $x=40$ gives a value exceeding $200$. The negative values of $x$ produce the corresponding positive values of $k$.
Thus there are $78$ candidate values from
$$x=\pm1,\pm2,\ldots,\pm39.$$
If two different integers $a,b$ produced the same $k$, then
$$5a+\frac{12}{a}=5b+\frac{12}{b},$$
which simplifies to $5ab=12$, impossible for distinct integers. Hence all $78$ values are distinct.

## Knowledge points

- Vieta-style root substitution
- Integer bounds
- Injectivity

## Key idea

Solve for the parameter $k$ in terms of the integer root $x$, then bound $x$.

## Common mistakes

- Counting only positive roots
- Assuming different roots must give different $k$ without proof
