# AMC12 2014-A Q22

![[assets/problems/2014-A/Q22.png]]

## Problem

The number $5^{867}$ is between $2^{2013}$ and $2^{2014}$. How many integer pairs $(m,n)$ satisfy $1\le m\le2012$ and
$$5^n<2^m<2^{m+2}<5^{n+1}?$$

## Answer

**B.** $279$

## Solution

Because $5^{867}$ lies between $2^{2013}$ and $2^{2014}$, the intervals
$$(5^n,5^{n+1})$$
contain either two or three powers of $2$. The desired chain occurs exactly when the interval contains three consecutive powers $2^m,2^{m+1},2^{m+2}$, and then $m$ is determined uniquely.
There are $867$ relevant intervals. If $d$ intervals contain two powers and $t$ contain three, then
$$d+t=867,\qquad2d+3t=2013.$$
Solving gives $t=279$, which is the number of valid pairs.

## Knowledge points

- Exponential inequalities
- Counting powers in intervals
- Linear systems

## Key idea

Translate the chain into a count of three consecutive powers of $2$ inside one interval between consecutive powers of $5$.

## Common mistakes

- Counting all intervals as valid
- Confusing the number of powers with the number of intervals
