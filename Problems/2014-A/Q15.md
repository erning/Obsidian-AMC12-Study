# AMC12 2014-A Q15

![[assets/problems/2014-A/Q15.png]]

## Problem

A five-digit palindrome has the form $abcba$, where $a\ne0$. Let $S$ be the sum of all five-digit palindromes. What is the sum of the digits of $S$?

## Answer

**B.** $18$

## Solution

Each palindrome is
$$10001a+1010b+100c.$$
There are $9$ choices for $a$ and $10$ choices for each of $b,c$. Therefore
$$\sum a=45\cdot10\cdot10,\qquad
\sum b=\sum c=45\cdot9\cdot10.$$
Hence
$$S=10001(4500)+1010(4050)+100(4050)=49{,}500{,}000.$$
The digit sum is
$$4+9+5=18.$$

## Knowledge points

- Place value
- Digit sums
- Counting

## Key idea

Sum each digit position independently, accounting for how often each digit occurs.

## Common mistakes

- Giving $a$ ten choices
- Summing the digit values without their place values
