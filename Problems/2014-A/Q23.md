# AMC12 2014-A Q23

![[assets/problems/2014-A/Q23.png]]

## Problem

The fraction
$$\frac1{99^2}=0.\overline{b_{n-1}b_{n-2}\cdots b_2b_1b_0},$$
where $n$ is the period length, is a repeating decimal. What is
$$b_0+b_1+\cdots+b_{n-1}?$$

## Answer

**B.** $883$

## Solution

Since $99^2=9801$, the repeating block can be read in two-digit base-$100$ blocks:
$$\frac1{99^2}=\frac1{(100-1)^2}.$$
The expansion contains the two-digit blocks from $00$ through $99$, except that $98$ is omitted and $99$ appears before the period repeats. If all blocks appeared, the digit sum would be
$$20(0+1+\cdots+9)=20\cdot45=900.$$
Removing the block $98$ subtracts $9+8=17$, so the required digit sum is
$$900-17=883.$$

## Knowledge points

- Repeating decimals
- Base-$100$ representation
- Digit sums

## Key idea

Group the decimal digits into two-digit blocks because the denominator is $(100-1)^2$.

## Common mistakes

- Treating the period as a base-10 block without structure
- Removing only one digit of the missing block
