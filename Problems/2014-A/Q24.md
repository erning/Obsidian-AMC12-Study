# AMC12 2014-A Q24

![[assets/problems/2014-A/Q24.png]]

## Problem

Let
$$f_0(x)=x+|x-100|-|x+100|,$$
and for $n\ge1$, let
$$f_n(x)=|f_{n-1}(x)|-1.$$
For how many values of $x$ is $f_{100}(x)=0$?

## Answer

**C.** $301$

## Solution

For $|x|\le100$,
$$f_0(x)=x+(100-x)-(x+100)=-x.$$
For $x<-100$, $f_0(x)=x+100-x-100=x+200$, and for $x\ge100$, $f_0(x)=x+x-100-(x+100)=x-200$.
Thus the range of $f_0$ is $[-100,100]$ and every value in this range has one or three preimages according to the piecewise-linear graph.
The recursion $y\mapsto|y|-1$ reaches $0$ at step $100$ exactly when
the initial value is $2k$ for an integer $-50\le k\le50$. Thus there are
$101$ target values. The endpoints $2k=\pm100$ have two preimages, while
the other $99$ target values have three preimages. Hence
$$99\cdot3+2\cdot2=301.$$

## Knowledge points

- Absolute values
- Piecewise-linear functions
- Preimage counting

## Key idea

Analyze the first function's graph, then count its preimages for the finite set of values that reach zero.

## Common mistakes

- Assuming every target has the same number of preimages
- Ignoring the endpoint behavior
