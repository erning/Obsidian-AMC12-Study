# AMC12 2012-A Q04

![[assets/problems/2012-A/Q04.png]]

## Problem

In a bag of marbles, $\frac35$ are blue and the rest are red. If the number of red marbles is doubled while the number of blue marbles stays the same, what fraction of the marbles will be red?

## Answer

**C.** $\frac47$

## Solution

Use $5$ marbles as a convenient scale: $3$ are blue and $2$ are red. After doubling the red marbles, there are $3$ blue and $4$ red marbles. The new total is $7$, so the red fraction is
$$\frac47.$$

## Knowledge points

- Fractions
- Ratios and proportions

## Key idea

Choose a total that realizes the given fraction, then update only the red count.

## Common mistakes

- Doubling the total number of marbles
- Keeping the original denominator $5$
