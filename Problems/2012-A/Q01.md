# AMC12 2012-A Q01

![[assets/problems/2012-A/Q01.png]]

## Problem

A bug crawls along a number line, starting at $-2$. It crawls to $-6$, then turns around and crawls to $5$. How many units does the bug crawl altogether?

## Answer

**E.** $15$

## Solution

The first leg has length
$$|-6-(-2)|=4,$$
and the second has length
$$|5-(-6)|=11.$$
Thus the total distance is
$$4+11=15.$$

## Knowledge points

- Absolute value
- Distance on a number line

## Key idea

Distance is the absolute value of the change in position on each leg.

## Common mistakes

- Computing the net displacement instead of total distance
- Ignoring the turn at $-6$
