# AMC12 2012-A Q08

![[assets/problems/2012-A/Q08.png]]

## Problem

An iterative average of $1,2,3,4,5$ is computed by arranging the numbers, averaging the first two, then averaging that result with the third, then with the fourth, and finally with the fifth. What is the difference between the largest and smallest possible values?

## Answer

**C.** $\frac{17}{8}$

## Solution

For an order $(a,b,c,d,e)$, the successive averaging gives
$$\frac{a+b+2c+4d+8e}{16}.$$
The later positions have larger weights, so the largest value uses the increasing order $(1,2,3,4,5)$:
$$\frac{1+2+6+16+40}{16}=\frac{65}{16}.$$
The smallest uses the reverse order $(5,4,3,2,1)$:
$$\frac{5+4+6+8+8}{16}=\frac{31}{16}.$$
The difference is
$$\frac{65-31}{16}=\frac{17}{8}.$$

## Knowledge points

- Weighted averages
- Rearrangement inequality
- Optimization

## Key idea

Rewrite the iterative process as a weighted sum and pair large values with large weights.

## Common mistakes

- Treating all five numbers as equally weighted
- Reversing which order gives the maximum
