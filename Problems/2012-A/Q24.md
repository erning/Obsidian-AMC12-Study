# AMC12 2012-A Q24

![[assets/problems/2012-A/Q24.png]]

## Problem

Let $\{a_k\}_{k=1}^{2011}$ be defined by $a_1=0.201$, $a_2=(0.2011)^{a_1}$, $a_3=(0.20101)^{a_2}$, and $a_4=(0.201011)^{a_3}$. More generally, for $k\ge2$, the base has $k+2$ digits after the decimal point: it is $0.20101\ldots0101$ when $k$ is odd and $0.20101\ldots01011$ when $k$ is even. Rearranging the sequence in decreasing order gives $\{b_k\}$. What is the sum of all $k$ such that $a_k=b_k$?

## Answer

**C.** $1341$

## Solution

Every base lies strictly between $0$ and $1$, so raising the base to a larger positive exponent decreases the value. Comparing consecutive terms gives the decreasing order
$$
1>a_2>a_4>\cdots>a_{2010}>a_{2011}>a_{2009}>\cdots>a_1>0.
$$

The even-indexed terms occupy the first part of the sorted sequence, followed by the odd-indexed terms in reverse order. A fixed position occurs in the odd tail when
$$2(k-1006)=2011-k.$$
Solving gives
$$3k=4023,qquad k=1341.$$
This is the only index with $a_k=b_k$, so the requested sum is $1341$.

## Knowledge points

- Exponential inequalities
- Monotonicity
- Permutation fixed points

## Key idea

Determine the complete order of the sequence, then locate indices that remain in the same position.

## Common mistakes

- Treating the bases as greater than $1$
- Sorting the odd-indexed terms in the wrong direction
