# AMC12 2009-B Q25

![[assets/problems/2009-B/Q25.png]]

## Problem

The set $G$ is defined by the points $(x, y)$ with integer coordinates, $3 \le |x| \le 7,$ and $3 \le |y| \le 7.$ How many squares of side at least $6$ have their four vertices in $G?$

(A) $125$
(B) $150$
(C) $175$
(D) $200$
(E) $225$

## Answer

**E.** $225$

## Solution

$G$ consists of four $5 \times 5$ blocks $G_1, \ldots, G_4,$ one in each quadrant. Any square of side $\ge 6$ uses exactly one vertex in each block, since two points in one block are less than $6$ apart while points in different blocks are at least $6$ apart.

Sliding each block inward by $(\pm 5, \pm 5)$ superimposes them on one $5 \times 5$ grid $G'$ (points with $|x|, |y| \le 2$). Each such square maps to either a single point of $G'$ or a square in $G'.$ So the count equals the number of points of $G'$ plus $4$ times the number of squares with vertices in $G'.$

A $5 \times 5$ grid has $25$ points and $50$ squares of all tilts, so the total is $25 + 4 \cdot 50 = 225.$

Thus, the correct answer is **E**.

## Knowledge points

- Lattice point
- Bijection
- Counting shapes in figures

## Key idea

Shift the four $5 \times 5$ blocks onto one grid; each valid square becomes a grid point or a grid square, so count both in a $5 \times 5$ lattice.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
