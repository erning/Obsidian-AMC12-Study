# AMC12 2017-A Q20

![[assets/problems/2017-A/Q20.png]]

## Problem

How many ordered pairs $(a,b)$ have $a>0$, $b$ an integer from 2 to 200, and

$$\bigl(\log_b a\bigr)^{2017}=\log_b(a^{2017})?$$

## Answer

**E.** $597$

## Solution

Let $u=\log_b a$. Since $\log_b(a^{2017})=2017u$, the equation becomes

$$u^{2017}=2017u.$$

Thus either $u=0$, giving $a=1$, or $u^{2016}=2017$, giving two real values of $u$ and hence two positive values of $a$ for every base. There are $199$ possible bases, so the total is

$$199+2(199)=597.$$

## Knowledge points

- Logarithms
- Odd/even powers

## Key idea

Replace the logarithm by a variable and count solutions for each base.

## Common mistakes

- Counting only the positive root of $u^{2016}=2017$.
