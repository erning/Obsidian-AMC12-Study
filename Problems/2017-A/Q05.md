# AMC12 2017-A Q05

![[assets/problems/2017-A/Q05.png]]

## Problem

Among 30 people, 20 know each other and 10 know no one. Acquaintances hug; non-acquaintances shake hands. How many handshakes occur?

## Answer

**B.** $245$

## Solution

Each of the 20 mutually acquainted people shakes hands with all 10 strangers, contributing $20\cdot10$ counts. Each stranger shakes hands with all other 29 people, contributing $10\cdot29$. Each handshake was counted twice, so

$$\frac{20\cdot10+10\cdot29}{2}=245.$$

## Knowledge points

- Double counting
- Combinatorial counting

## Key idea

Count handshakes from both groups, then divide by two.

## Common mistakes

- Counting only stranger-stranger handshakes.
