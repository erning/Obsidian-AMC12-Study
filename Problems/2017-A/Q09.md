# AMC12 2017-A Q09

![[assets/problems/2017-A/Q09.png]]

## Problem

Let $S$ be the set of points $(x,y)$ for which two of $3$, $x+2$, and $y-4$ are equal and the third is no greater than their common value. Which description is correct?

## Answer

**E.** Three rays with a common endpoint

## Solution

If $3=x+2\ge y-4$, then $x=1$ and $y\le7$, a downward ray. If $3=y-4\ge x+2$, then $y=7$ and $x\le1$, a leftward ray. If $x+2=y-4\ge3$, then $y=x+6$ and $x\ge1$, a diagonal ray. All three start at $(1,7)$.

## Knowledge points

- Piecewise geometric conditions
- Coordinate geometry

## Key idea

Enumerate which pair is the common larger value.

## Common mistakes

- Using only equality and forgetting the inequality condition.
