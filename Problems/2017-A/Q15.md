# AMC12 2017-A Q15

![[assets/problems/2017-A/Q15.png]]

## Problem

Let $f(x)=\sin x+2\cos x+3\tan x$. In which interval does the smallest positive root lie?

## Answer

**D.** $(3,4)$

## Solution

For $0<x<\pi/2$, all terms are positive. For $\pi/2<x<\pi$, the tangent term is negative and the function remains negative until it reaches $x=\pi$, where $f(\pi)=-2$. At $x=5\pi/4$, $\tan x=1$ and

$$f(5\pi/4)=-\frac{\sqrt2}{2}-\sqrt2+3>0.$$

By continuity, the first positive root lies in $(\pi,5\pi/4)\subset(3,4)$.

## Knowledge points

- Trigonometric signs
- Intermediate Value Theorem

## Key idea

Bracket the first root using sign changes rather than solve the equation.

## Common mistakes

- Forgetting the tangent discontinuity at $\pi/2$.
