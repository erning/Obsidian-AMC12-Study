# AMC12 2017-A Q21

![[assets/problems/2017-A/Q21.png]]

## Problem

Start with $S=\{0,10\}$. Whenever an integer is a root of a polynomial whose coefficients all lie in $S$, add it to $S$. When no more elements can be added, how many elements does $S$ contain?

## Answer

**D.** $9$

## Solution

The polynomial $10x+10$ adds $-1$. A polynomial with coefficients in the evolving set adds $1$ and $-10$. Then $x^3+x-10$ adds 2, and $x+2$ adds $-2$. Finally $2x-10$ and $2x+10$ add $5$ and $-5$. Thus

$$S=\{0,\pm1,\pm2,\pm5,\pm10\}.$$

No other integer can occur: by the Rational Root Theorem, an integer root divides the constant coefficient, and every possible constant coefficient is a factor of 10. Hence there are 9 elements.

## Knowledge points

- Rational Root Theorem
- Closure processes

## Key idea

Generate attainable roots, then use divisibility to prove termination.

## Common mistakes

- Assuming every divisor of 10 must automatically be generated.
