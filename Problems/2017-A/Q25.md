# AMC12 2017-A Q25

![[assets/problems/2017-A/Q25.png]]

## Problem

The vertices of a centrally symmetric hexagon in the complex plane are

$$\left\{\sqrt2i,-\sqrt2i,\frac{1+i}{\sqrt8},\frac{-1+i}{\sqrt8},\frac{1-i}{\sqrt8},\frac{-1-i}{\sqrt8}\right\}.$$

For each of 12 independent choices, select an element and let $P$ be their product. What is the probability that $P=-1$?

## Answer

**E.** $\frac{2^2\cdot5\cdot11}{3^{10}}$

## Solution

Two choices have magnitude $\sqrt2$, and four have magnitude $1/\sqrt8$. If $k$ selections come from the first group, then

$$|P|=(\sqrt2)^k(1/\sqrt8)^{12-k}.$$

For $|P|=1$, we need $k=8$, so exactly four selections come from the second group. The probability of this pattern is

$$\binom{12}{8}\left(\frac13\right)^8\left(\frac23\right)^4=\frac{880}{3^{10}}.$$

For any such pattern, the possible arguments are equally distributed among $1,-1,i,-i$, so one quarter give $P=-1$. Thus

$$P(P=-1)=\frac14\cdot\frac{880}{3^{10}}=\frac{220}{3^{10}}=\frac{2^2\cdot5\cdot11}{3^{10}}.$$

## Knowledge points

- Complex modulus and arguments
- Binomial probability

## Key idea

First force modulus 1, then impose the argument condition.

## Common mistakes

- Counting the six values as equally likely magnitude classes.
- Forgetting the final one-quarter argument condition.
