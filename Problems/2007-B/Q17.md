# AMC12 2007-B Q17

![[assets/problems/2007-B/Q17.png]]

## Problem

If $a$ is a nonzero integer and $b$ is a positive number such that $ab^2=\log_{10}b,$ what is the median of the set $\{0,1,a,b,1/b\}?$

(A) $0$
(B) $1$
(C) $a$
(D) $b$
(E) $\dfrac{1}{b}$

## Answer

**D.** $b$

## Solution

For $b\gt0$, we have $b\lt10^b$, hence $\log_{10}b\lt b$. If $b=1$, then $a=0$. If $b\gt1$, then $0\lt\dfrac{\log_{10}b}{b^2}\lt1$, so $a$ could not be an integer.

Hence $0\lt b\lt1,$ so $\log_{10}b\lt0$ and $a=\dfrac{\log_{10}b}{b^2}\lt0.$ Thus $a\lt0\lt b\lt1\lt\dfrac1b,$ and the middle value of the sorted set is $b.$

Thus, the correct answer is **D**.

## Key idea

Use logarithm rules and translate the condition into an inequality.
