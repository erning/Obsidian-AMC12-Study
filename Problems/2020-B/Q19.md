# AMC12 2020-B Q19

![[assets/problems/2020-B/Q19.png]]

## Problem

The four transformations $L$, $R$, $H$, and $V$ rotate a square $90^\circ$ counterclockwise, rotate it $90^\circ$ clockwise, reflect it across a horizontal axis, and reflect it across a vertical axis, respectively. A sequence of $20$ transformations is chosen from these four transformations. How many sequences return the square's labels to their original positions?

## Answer

**C.** $2^{38}$

## Solution

The transformations form the eight-element symmetry group of a square. Let $f_k(g)$ be the number of length-$k$ sequences whose combined transformation is $g$. Starting with $f_0(I)=1$ and all other values $0$, update by

$$f_{k+1}(g)=f_k(gL^{-1})+f_k(gR^{-1})+f_k(gH^{-1})+f_k(gV^{-1}).$$

This is a finite recurrence on the eight symmetries, so it counts all $4^{20}$ sequences without listing them. Iterating the recurrence to $k=20$ gives

$$f_{20}(I)=2^{38}.$$

The identity state is exactly the state in which all labels return to their original positions, so the answer is $2^{38}$.

## Knowledge points

- Symmetry group of a square
- Recurrence and state counting
- Composition of transformations

## Key idea

Track only the current symmetry of the square. The eight possible states make a short dynamic-counting problem.

## Common mistakes

- Treating $L$ and $R$ as unrelated numerical operations instead of inverse rotations.
- Counting only sequences with total rotation $0$ and ignoring reflections.
- Forgetting that a reflection followed by another reflection can be a rotation.
