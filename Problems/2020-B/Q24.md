# AMC12 2020-B Q24

![[assets/problems/2020-B/Q24.png]]

## Problem

Let $D(n)$ be the number of ordered lists of integers $(f_1,f_2,\ldots,f_k)$ such that $f_i>1$ and $f_1f_2\cdots f_k=n$. What is $D(96)$?

## Answer

**A.** $112$

## Solution

Set $D(1)=1$, representing the empty product. For $n>1$, choose the first factor $d$. The remaining ordered list is a factorization of $n/d$, so

$$D(n)=\sum_{\substack{d\mid n\\d>1}}D\left(\frac nd\right).$$

Using the recurrence for the divisors needed for $96=2^5\cdot3$ gives

$$
\begin{aligned}
D(2)&=D(3)=1, &D(4)&=2, &D(6)&=3,\\
D(8)&=4, &D(12)&=8, &D(16)&=8,\\
D(24)&=20, &D(32)&=16, &D(48)&=48.
\end{aligned}
$$

Therefore

$$\begin{aligned}
D(96)&=D(48)+D(32)+D(24)+D(16)+D(12)+D(8)\\
&\quad+D(6)+D(4)+D(3)+D(2)+D(1)\\
&=48+16+20+8+8+4+3+2+1+1+1\\
&=112.
\end{aligned}$$

## Knowledge points

- Ordered multiplicative partitions
- Divisor recurrence
- Recursive counting

## Key idea

Condition on the first factor in the ordered factorization.

## Common mistakes

- Treating the factors as unordered.
- Omitting $D(1)$ when the first factor equals $n$.
- Allowing a factor equal to $1$.
