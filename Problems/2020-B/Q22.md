# AMC12 2020-B Q22

![[assets/problems/2020-B/Q22.png]]

## Problem

What is the maximum value, over all real numbers $t$, of

$$\frac{(2^t-3t)t}{4^t}?$$

## Answer

**C.** $\frac1{12}$

## Solution

Since $4^t=(2^t)^2$, rewrite the expression as

$$\frac{2^t\,t-3t^2}{(2^t)^2}
=\frac{t}{2^t}-3\left(\frac{t}{2^t}\right)^2.$$

Let $u=t/2^t$. The expression is $u-3u^2$, a downward-opening parabola. Its maximum occurs at

$$u=\frac16,$$

with value

$$\frac16-3\left(\frac16\right)^2=\frac1{12}.$$

The value $u=1/6$ is attained by some real $t$, so this upper bound is achievable.

## Knowledge points

- Algebraic substitution
- Quadratic optimization
- Completing the square or vertex form

## Key idea

The complicated exponential expression depends only on $u=t/2^t$.

## Common mistakes

- Differentiating the original expression unnecessarily.
- Maximizing $t/2^t$ instead of the quadratic in that quantity.
