# AMC12 2020-B Q06

![[assets/problems/2020-B/Q06.png]]

## Problem

For all integers $n\ge9$, the value of

$$\frac{(n+2)!-(n+1)!}{n!}$$

is always which of the following?

## Answer

**D.** A perfect square

## Solution

Factor $(n+1)!$ from the numerator:

$$\frac{(n+2)!-(n+1)!}{n!}
=\frac{(n+1)!((n+2)-1)}{n!}
=\frac{(n+1)!(n+1)}{n!}
=(n+1)^2.$$

This is a perfect square for every integer $n$ in the stated range.

## Knowledge points

- Factorials
- Algebraic factorization

## Key idea

Rewrite $(n+2)!$ as $(n+2)(n+1)!$ before dividing by $n!$.

## Common mistakes

- Treating $(n+2)!-(n+1)!$ as $(n+1)!$.
- Choosing “perfect cube” because factorials contain many factors.
