# AMC12 2009-A Q21

![[assets/problems/2009-A/Q21.png]]

## 题目

设 $p(x) = x^3 + ax^2 + bx + c,$ 其中 $a,$ $b,$ $c$ 为复数。已知 $$ \begin{gathered} p(2009 + 9002\pi i) \\ = p(2009) \\ = p(9002) = 0. \end{gathered} $$ 多项式 $x^{12} + ax^8 + bx^4 + c$ 的非实零点有多少个？

(A) $4$
(B) $6$
(C) $8$
(D) $10$
(E) $12$

## 答案

**C.** $8$

## 解答

由于 $x^{12} + ax^8 + bx^4 + c = p(x^4),$ 一个数为零点当且仅当其四次方等于 $p$ 的某个根，即 $2009 + 9002\pi i,$ $2009,$ 或 $9002.$

方程 $x^4 = 2009 + 9002\pi i$ 有四个不同的非实根。$x^4 = 2009$ 和 $x^4 = 9002$ 各有两个实根和两个非实根。

因此非实零点个数为 $4 + 2 + 2 = 8.$

故正确答案为 **C**。

## 知识点

- 多项式
- 复数
- 单位根

## 关键思路

将原多项式写为 $p(x^4)$，再分别统计每个根对应的四次方根。

## 常见错误

- 未先识别相关数量和条件就套用公式。
- 计入不符合条件的情形，或忽略题目中的边界条件。
