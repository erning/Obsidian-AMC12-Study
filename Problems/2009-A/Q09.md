# AMC12 2009-A Q09

![[assets/problems/2009-A/Q09.png]]

## Problem

Suppose that $f(x + 3) = 3x^2 + 7x + 4$ and $f(x) = ax^2 + bx + c.$ What is $a + b + c?$

(A) $-1$
(B) $0$
(C) $1$
(D) $2$
(E) $3$

## Answer

**D.** $2$

## Solution

Note that $a + b + c = f(1).$

Using $f(x + 3) = 3x^2 + 7x + 4$ with $x = -2$ gives $$ \begin{aligned} f(1) &= f(-2 + 3) \\ &= 3(-2)^2 + 7(-2) + 4 \\ &= 12 - 14 + 4 = 2. \end{aligned} $$

Thus, the correct answer is **D**.

## Knowledge points

- Function
- Substitution

## Key idea

Replace $x$ by $x+3$ in the first equation to obtain $f(x)$, then compare coefficients with $ax^2+bx+c$.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
