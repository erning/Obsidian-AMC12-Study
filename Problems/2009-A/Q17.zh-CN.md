# AMC12 2009-A Q17

![[assets/problems/2009-A/Q17.png]]

## 题目

设 $a + ar_1 + ar_1^2 + ar_1^3 + \cdots$ 与 $a + ar_2 + ar_2^2 + ar_2^3 + \cdots$ 是两个首项相同、各项均为正数且彼此不同的无穷等比级数。第一个级数的和为 $r_1,$ 第二个级数的和为 $r_2.$ 求 $r_1 + r_2$。

(A) $0$
(B) $\frac12$
(C) $1$
(D) $\frac{1+\sqrt5}{2}$
(E) $2$

## 答案

**C.** $1$

## 解答

对于首项为 $a$、公比为 $r$ 的级数，其和为 $\frac{a}{1 - r} = r,$ 因此 $r^2 - r + a = 0.$

$r_1$ 和 $r_2$ 均满足这个二次方程。两个级数不同，所以 $r_1 \ne r_2,$ 即它们是该方程的两个不同根。由韦达定理，$r_1 + r_2 = 1.$

故正确答案为 **C**。

## 知识点

- 等比数列
- 韦达定理

## 关键思路

将两个无穷等比级数的和与公比关联起来，并利用韦达定理。

## 常见错误

- 未先识别相关数量和条件就套用公式。
- 计入不符合条件的情形，或忽略题目中的边界条件。
