# AMC12 2009-A Q18

![[assets/problems/2009-A/Q18.png]]

## Problem

For $k \gt 0,$ let $I_k = 10\ldots064,$ where there are $k$ zeros between the $1$ and the $6.$ Let $N(k)$ be the number of factors of $2$ in the prime factorization of $I_k.$ What is the maximum value of $N(k)?$

(A) $6$
(B) $7$
(C) $8$
(D) $9$
(E) $10$

## Answer

**B.** $7$

## Solution

Note that $I_k = 10^{k+2} + 64$ $= 2^{k+2}5^{k+2} + 2^6.$

For $k \lt 4$ the first term has fewer than $6$ factors of $2,$ so $N(k) \lt 6.$ For $k \gt 4$ the first term is divisible by $2^7$ but the $2^6$ term is not, so $N(k) \lt 7.$

For $k = 4,$ $I_4 = 2^6(5^6 + 1).$ Since $5^6 + 1$ $= (5^2 + 1)\big((5^2)^2 - 5^2 + 1\big)$ $= 26\cdot 601,$ and $26 = 2\cdot 13$ contributes exactly one more factor of $2,$ we get $N(4) = 7.$

So the maximum value is $7.$

Thus, the correct answer is **B**.

## Knowledge points

- Prime factorization
- Sum and difference of cubes

## Key idea

Factor $10\ldots064$ as a difference of cubes and use the prime-factorization condition to determine the number of digits.

## Common mistakes

- Applying a formula before identifying the relevant quantities and constraints.
- Counting invalid cases or overlooking a boundary condition from the statement.
