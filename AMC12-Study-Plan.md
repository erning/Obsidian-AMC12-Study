# AMC 12 Study Notes Plan

## Goal

Build complete AMC 12 study notes from 2000 onward. Every administered exam
version gets its own directory under Problems/ and assets/problems/.

## Note structure

Each exam directory is named <year>-<part> and contains:

- Overview.md
- Q01.md through Q25.md

Each matching asset directory contains:

- the full problem PDF
- one accurately cropped question image for each problem

Every question note must include:

- the problem image at the top
- the complete problem statement
- the correct answer
- a detailed solution
- knowledge points
- key idea
- common mistakes
- related problems
- source links

All mathematical expressions must use Obsidian-compatible $...$ or $$...$$
syntax. File names remain English and use the established capitalization.

## Workflow for each exam

1. Verify the official problem PDF, answer key, and a reliable solution source.
2. Create the mirrored Problems/<year>-<part>/ and
   assets/problems/<year>-<part>/ directories.
3. Download the PDF and crop each question precisely.
4. Write the overview, answer key, and all 25 detailed question notes.
5. Check problem numbering, answers, image embeds, internal links, and LaTeX.
6. Commit only that exam's notes and assets.
7. Update this plan's progress after the exam is complete.

## Commit rule

Use one English ASCII Conventional Commit for each exam. Do not combine two
exam versions in one commit. This temporary plan file may be deleted after the
historical collection is complete.
